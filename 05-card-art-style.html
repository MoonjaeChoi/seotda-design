K-Sutda — 화투 카드 아트 스타일 가이드

02-asset-spec.md §1 의 화투 40장 작업을 위한 상세 스타일 가이드. 톤·색·필치·식물 묘사 디테일.

0. 작업 전제

0-1. 화투 = 한국화 (韓國畵)

본 게임의 화투는 한국식. 일본 하나후다 (花札) 와 구분 필요:

일본 하나후다 K-Sutda 화투 (한국식)
색감 채도 높음, 강한 원색 차분, 한지·먹·금 위주
필치 평면적, 그래픽 한국 전통 민화 + 모던
식물 묘사 일본식 (벚꽃 작고 가늘) 한국식 (송학 굵직, 단풍 풍성)
인물·동물 일본식 학·사슴 (가는 라인) 한국 토착 학·사슴 (단단함)

참고 자료: 단청, 책가도, 화조도, 조선 민화

0-2. 게임 내 카드 크기

  • 화면 표시: 220 × 320 px (모바일 ×2 retina 440×640)
  • 실제 그릴 때: 600 × 900 px 권장 (다운스케일이 더 깔끔)
  • 안전 영역: 가장자리 12px margin (잘림 방지)

0-3. UI 통합 영역

┌──────────────────┐
│ 1월              │ ← 좌상단: 월 숫자 (12px, 흰색 글자, 자동 overlay)
│                  │
│      ┌─┐         │
│      │ │         │
│  메인 일러스트   │ ← 중앙: 식물·동물 일러스트 (이 영역에 집중)
│      │ │         │
│      └─┘         │
│              광  │ ← 우하단: type 한글 (12px, 자동 overlay)
└──────────────────┘

→ 좌상단 / 우하단 영역은 비워두거나 옅게 처리. UI 가 자동 overlay.

1. 컬러 팔레트 (월별)

각 월의 식물 + 계절감을 반영한 핵심 색. 카드 상단의 액센트 바 색으로 사용 (Card.gd._draw_front 의 accent_color).

월 식물 액센트 색 (HEX) 분위기
1월 송학 (소나무·학) #2E6B47 짙은 솔잎 녹 새해, 정월
2월 매조 (매화) #C72E4F 매화 핑크 이른 봄
3월 벚꽃 #F2A6BD 벚꽃 분홍 봄
4월 흑싸리 (꾀꼬리) #2E3E5C 짙은 청남 새벽
5월 난초 #F5E68A 연한 노랑 늦봄
6월 모란·나비 #9C2E5C 진홍 모란 여름
7월 홍싸리·멧돼지 #B5462E 붉은 황토 한여름
8월 공산·달·기러기 #2E3556 짙은 밤색 가을 보름
9월 국화 #E8A52E 황금 국화 가을
10월 단풍·사슴 #A03A1F 단풍 적색 늦가을

규칙:

  • 액센트 바는 카드 상단 16px 영역
  • 하단 16px 는 type 색 (광=금 / 띠=주홍 / 열=청 / 피=회색) — Card.gd._type_color() 참조

2. Type 별 디자인 패턴

화투는 한 월에 4 type. 각 type 의 의미·시각 다름.

2-1. 광 (光) — 총 3장 (1월 / 3월 / 8월)

의미: 최강패. 광땡 (3월광 + 8월광) = jokbo 100.

디자인 원칙:

  • 가장 화려하고 정성스러운 일러스트
  • 금색 글로우 보더 (is_gwang_special 인 1·3·8월 광은 자동 4px 금색 보더)
  • 일러스트 자체에도 금색 액센트 (학의 부리, 달, 보름달 광원 등)
  • 한 마디로 "이 카드만 다른 카드 보다 비주얼적으로 압도적"

예시 일러스트:

  • 1월 송학광: 푸른 솔잎 + 학 1마리 + 떠오르는 해 (붉은 해 + 금색 광원)
  • 3월 벚꽃광: 만개한 벚꽃 + 검은 만막 (장막) + 달
  • 8월 공산광: 공산 (검은 산) + 보름달 (큰 금색 원) + 기러기 3마리

2-2. 띠 (帶) — 각 월 1장

의미: 중간 가치. 띠 (가로 띠) 가 있는 카드.

디자인 원칙:

  • 메인 일러스트 위에 붉은 띠 또는 명문 띠 가로로 깔림
  • 1·2·3월 띠는 붉은 띠 (홍단), 4월 띠는 청, 등 월별 다름 (한국 화투 규정 참조)
  • 띠는 정확한 위치 (카드 상단~중단 사이) 가로 폭 80%

예시: 2월 매조띠 — 매화 + 새 + 가로 명문 띠

2-3. 열 (列) — 각 월 1장

의미: 중간 가치. 동물 또는 특수 객체.

디자인 원칙:

  • 동물 (학·꾀꼬리·나비·기러기·사슴 등) 또는 인물
  • 한국 토착 동물 표현 (예: 한국 학 — 단정학)
  • 색상: 동물 위주 + 식물은 배경

2-4. 피 (皮) — 각 월 1~2장 (월별 다름)

의미: 가장 흔한 카드. 점수에 큰 기여 없음.

디자인 원칙:

  • 식물만 (동물·띠 없음)
  • 가장 단순한 일러스트
  • 같은 월의 다른 type 과 시각 통일성 (식물 묘사 일관)

3. 월별 일러스트 상세 가이드

각 월의 핵심 비주얼 모티프. 디자이너가 첫 시안 작업 시 참고.

1월 — 송학 (松鶴)

모티프: 소나무 + 학 + 떠오르는 해
색감: 짙은 녹 + 흰색 (학) + 붉은 점 (학 머리)
특징: 굵고 단단한 솔잎 (한국 적송 — 일본 흑송과 다름)
한국적 디테일: 학은 한국 단정학 — 흰 몸 + 검은 날개 끝

2월 — 매조 (梅鳥)

모티프: 매화 + 매조 (참새 또는 휘파람새)
색감: 분홍 매화 + 진한 가지 + 회색 새
특징: 매화 꽃잎은 5장, 끝 둥글게 (일본 매화는 끝이 뾰족)

3월 — 벚꽃 + 만막

모티프: 활짝 핀 벚꽃 + 만막 (검은 막)
색감: 분홍 (벚꽃) + 짙은 검정 (만막) + 광패는 금색 보름달
특징: 만막은 일본 사쿠라 막 같은 평면적 처리 ❌ → 한국식 비단 막 (음영 강조)
광패 강조: 보름달 큰 금색 원 + 광원 글로우

4월 — 흑싸리 (黑싸리)

모티프: 등나무 (검은 가지) + 꾀꼬리
색감: 짙은 청남 + 노란 꾀꼬리
특징: 한국 화투에서 4월은 어두운 톤 — 약패의 분위기
독사 (1월+4월) 공유 → 1월 송학과 시각적 대비 (밝음 vs 어두움)

5월 — 난초

모티프: 난초 + (열 카드에) 다리 / 새
색감: 연한 노랑 + 회녹색 잎
특징: 한국 동양란 (잎이 길고 가늘) — 일본 lan 과 비슷하지만 더 단정

6월 — 모란 + 나비

모티프: 활짝 핀 모란 + 나비
색감: 진홍 모란 + 흰/검 나비
특징: 모란은 한국 민화의 단골 — 큼직하게, 풍성하게

7월 — 홍싸리 + 멧돼지

모티프: 홍싸리 (붉은 등나무) + 멧돼지
색감: 붉은 갈색 + 짙은 멧돼지
특징: 멧돼지는 한국 야생 멧돼지 — 굵고 단단한 체형

8월 — 공산·달·기러기

모티프: 공산 (검은 산 실루엣) + 보름달 + 기러기 3마리
색감: 짙은 밤 (남청) + 금색 달 + 흰 기러기
특징: 광패 — 가을밤의 정취. 달이 카드의 중심
한국적 디테일: 공산은 한국식 비현실적 산 (한국 산수화 톤)
광패 강조: 달은 카드의 1/3 차지하는 큰 금색 원

9월 — 국화

모티프: 만개한 국화 + (열 카드에) 술잔
색감: 황금 국화 + 짙은 잎
특징: 한국 가을 국화 — 풍성한 꽃잎 (일본 키쿠와 다름)
9땡 + 구삥 공유 카드

10월 — 단풍·사슴

모티프: 단풍나무 + 사슴
색감: 붉은 단풍 + 갈색 사슴
특징: 한국 노루·사슴 — 단단한 체격, 황금빛
장땡 + 장삥 공유 → 핵심 카드 중 하나

4. 카드 뒷면 디자인

4-1. 현재 코드 (placeholder)

Card.gd._draw_back:

  • 짙은 녹색 (#1A4D33) 베이스
  • 격자 패턴 (10px 간격, 옅은 선)
  • 중앙 원형 마크 (작은 녹색 원)
  • 짙은 보더

4-2. 디자이너 작업 요청

컨셉: 한국 전통 문양 (구름·매화·학 중 1)

옵션 A — 구름

  • 짙은 녹 (#0D2619 BG_INK) 또는 짙은 자주 베이스
  • 한국 전통 운문 (雲紋) — 단청 책가도의 구름
  • 중앙에 작은 K-Sutda 로고 (옵션)

옵션 B — 매화

  • 짙은 베이스 + 옅은 매화 가지 패턴
  • 가지는 모서리에서 중앙으로 뻗는 형태
  • 5장 매화 꽃 — 한국식 둥근 꽃잎

옵션 C — 학 (단순화)

  • 짙은 베이스 + 흰 학 실루엣 (중앙 또는 모서리)
  • 미니멀, 모던 한국풍

스펙:

  • 220×320 px (카드와 동일)
  • PNG 단일 파일 (변형 1종)
  • 광패 보더 (금색 4px) 는 코드가 자동 적용 (뒷면엔 적용 안 됨 — 게임 진행 중 광패 인식 노출 위함)

5. 일러스트 톤·필치

5-1. 권장 톤

  • 한국 민화 + 약간 모던 균형
  • 색감 차분 (한지·먹·자연 색)
  • 그라데이션 OK (하늘·달 광원 등) 단, 과하지 않게
  • 라인: 굵은 먹선 + 부드러운 채색

5-2. 피해야 할 톤

  • ❌ 픽셀 아트 (발라트로 톤)
  • ❌ 일본 우키요에 그대로 모방
  • ❌ 카툰·만화 톤 (귀여움)
  • ❌ 사진 합성 / 디지털 페인팅 너무 사실적
  • ❌ 형광 컬러
  • ❌ 일본 화투 카드 직접 트레이싱

5-3. 권장 작업 도구

  • Adobe Photoshop / Procreate — 디지털 페인팅
  • Adobe Illustrator — 벡터 (확장성 ↑, 단 한국화 톤 어려움)
  • Clip Studio Paint — 만화 / 일러스트 (한국 일러스트레이터 다수 사용)

PSD / PROCREATE / AI 원본 파일 함께 납품 권장.

6. 광땡 강조 — 특수 처리

3월광 + 8월광 = 광땡 (jokbo 100, 게임 내 최강).

6-1. 코드 자동 처리

Card.gd 에서 is_gwang_special (1·3·8월 광) 검사 후:

  • 보더 색 → 금색 (#F3C72E)
  • 보더 두께 → 2px → 4px

6-2. 디자이너 추가 권장

광패 일러스트 자체에도 차별화:

  • 3월 벚꽃광: 보름달 추가 + 만막에 금색 자수 패턴
  • 8월 공산광: 보름달이 카드 1/3 크기 (다른 카드 달보다 1.5배)
  • 1월 송학광: 떠오르는 해 + 학 부리에 금색

이 차이로 사용자가 "광패다!" 즉각 인지.

7. 작업 순서 (Tier 별)

Tier 1 (먼저)

  1. 1월 송학광 샘플 1장 (개발자 톤 합의용)
  2. 3월 벚꽃광 샘플 1장 (광땡 핵심)
  3. → 톤 합의 → 양산 진행

Tier 2

  1. 8월 공산광 (광패 3장 완성)
  2. 광패 이외 1·3·8월 각 type (3 × 3 = 9장)

Tier 3

  1. 1, 2, 9, 10월 — 특수 족보 (알리·구삥·장삥 등) 관련 카드
  2. 10월 단풍·사슴 (장땡)

Tier 4

  1. 나머지 (2·4·5·6·7월) 일반 카드

Tier 5

  1. 카드 뒷면 (3 옵션 중 1 확정 후 양산)

8. QA 체크리스트 (디자이너 self-check)

  • 220×320 px (또는 600×900 원본)
  • PNG 투명 배경
  • 좌상단 12px / 우하단 12px 안전 영역 (UI overlay 자리)
  • 액센트 색 (월별) 사용 — 카드 상단 16px
  • type 색 (광/띠/열/피) — 카드 하단 16px
  • 광패면 금색 글로우 / 보름달 / 햇무리 등 추가 강조
  • 한국 토착 식물·동물 (일본 톤 ❌)
  • 차분한 색감 (형광 ❌)
  • 라이선스 명시 (work-for-hire 또는 본인 창작)

9. 라이선스 / 저작권

  • 한국 전통 화투 자체는 public domain (수백 년 됨)
  • 특정 출판사 (손오공 / 한게임 등) 의 카드 디자인은 저작권 보유 — 직접 트레이싱 금지
  • 모든 일러스트는 본인 새로 그리기 (참고 OK, 모방 ❌)
  • 납품 시 origin 명시 (예: "본인 창작, 한국 전통 화투 참고")
