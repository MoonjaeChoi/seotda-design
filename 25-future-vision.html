1. Phase 4 — 다인 플레이 / 새 게임 모드 (출시 후 6개월)

1-1. 다인 플레이 모드 (Multiplayer)

K-Sutda 의 본질 — 섯다는 본래 4~6인 게임. AI 1명 → 다인 환경 도입.

디자인 옵션

옵션 장점 단점
로컬 다인 (같은 PC / 모바일 핫시트) 빠른 개발, 친구 끼리 글로벌 X
온라인 다인 (P2P) 글로벌 가능, 단순 인프라 NAT / 매칭 어려움
온라인 다인 (전용 서버) 안정적 매칭, 큰 커뮤니티 인프라 비용 / 운영
비동기 다인 (시드 챌린지) 인프라 ↓, 결정성 활용 실시간 X

🎯 추천 — 단계적:

  1. Phase 4.1 로컬 핫시트 (2~4인) — 친구 모임 / 동네 노름판
  2. Phase 4.2 온라인 P2P (2인) — 글로벌 1:1
  3. Phase 5+ 전용 서버 매칭 — 큰 규모 (옵션)

게임 디자인 변화

2인 → 4인 다인:
- 포지션 (선/후위) 회전 — 매 판 다른 사람이 선
- 정보 비대칭 — 다른 플레이어의 베팅에서 패 추측
- 메모리 게임 — 누가 어떤 패턴인지 학습
- 블러프 의미 ↑ — 여러 명 속이기

기존 솔로 메커니즘 유지:
- 마패 카드 (각자 보유)
- mulligan
- 2단계 베팅 (1차 / 2차)
- 명성치 한도

캐릭터 / 아바타

  • 사용자 닉네임 + 캐릭터 아바타 (5~10 종 선택)
  • 한국 도박판 톤 (한복 / 정장 / 갓 / 탈)
  • 베타 진행하며 추가 아바타 unlock (메타 진행)

인프라 / 비용

  • 로컬: 코드만 (인프라 X)
  • 온라인 P2P: STUN/TURN 서버 (월 $50~100)
  • 전용 서버: 클라우드 (Supabase + Game Server, 월 $200~500)

1-2. 새 게임 모드

Mode 1 — Endless (무한 모드)

8 단계 클리어 후 계속:
- 단계 9, 10, 11... 무한
- AI 패턴 더 강력 (보스 부스트 ↑)
- 마패 추가 효과 (Legendary 더 자주)
- 명성치 한도 ↓ (-100 → -50, 더 어려움)
- 리더보드 — 가장 깊은 단계 도달

대상: P1 코어 / P6 스트리머 (도전 의지)

Mode 2 — Daily Challenge (일일 챌린지)

매일 새 시드 (전 세계 동일):
- 모든 사용자 같은 카드 / 같은 AI
- 마패 빌드는 자기 자유
- 24시간 내 가장 높은 점수
- 글로벌 리더보드 (Supabase)
- 한국 시간 자정 갱신

대상: 모든 페르소나 (특히 P5 캐주얼 / P6 스트리머)

Mode 3 — Speedrun

가장 빠른 단계 8 클리어:
- 타이머 표시
- 시드 자유
- 글로벌 리더보드 (시간 순)

대상: P6 스트리머 (영상 콘텐츠)

Mode 4 — Hardcore (영구사망)

한 번 패배하면 끝:
- 게임오버 한도 -100 (더 엄격)
- 자동 세이브 X (수동 X)
- 단계 8 클리어 → "최고 영광" 칭호
- 베타 후 도입

대상: P1 코어

1-3. 메타 진행 (Persistent Progression)

게임 끝나도 사라지지 않는 진행:

[칭호 시스템]
- "전설" (단계 8 클리어)
- "탈춤꾼" (블러프 100회 성공)
- "광의 신" (광땡 50회)
- "구라쟁이" (게임오버 50회 — 좌절 유머)
- ...

[아바타 unlock]
- 단계 8 클리어 1회 → 갓 무명
- 5회 → 한복 무명
- 10회 → 황금 갓 무명
- ...

[배경 unlock]
- 단계 5 클리어 → 사찰 배경
- 단계 8 클리어 → 다실 배경
- ...

[누적 통계 표시]
- 총 런 / 클리어 / 광땡 / 블러프
- 가장 강한 빌드 (TOP 3)
- 누적 명성치 변화

  • ✅ 사행성 모사 회피 + 윤리적 디자인
  • ✅ 솔로 개발자 → 작은 팀 / IP 회사 진화
  • ❌ 단순 모방 (발라트로 카피)
  • ❌ Pay-to-Win 으로 전환

8-2. 외부 제안 / 컬래버

- 다른 게임과의 컬래버: 한국 정체성 보존 시 OK
- 영화 / 드라마 IP 사용: 신중 (한국 도박 콘텐츠 — 사회적 영향)
- 한국 정부 사업 (콘텐츠 진흥원 등): 가능 (자금 / 노출)
- 광고 / 협찬: 인디 톤 유지 (과한 광고 X)

8-3. 후속작 결정 기준

정식 출시 후 12~18개월:
- 사용자 retention ≥ 30% (3개월)
- 활성 MAU ≥ 1,000명
- Discord 커뮤니티 ≥ 500명
- 인플루언서 자연 영상 ≥ 50개
- 누적 판매 ≥ 30,000 카피

위 모두 만족 시 → 후속작 결정.
부족 시 → 라이브 운영 / DLC 집중.

9. 위기 대응 / 출구 전략

9-1. 실패 시나리오

[시나리오 1: 출시 후 6개월 판매 < 5,000]
- 마케팅 전략 재검토
- 작은 무료 업데이트 + Steam Sale
- 인플루언서 재배포
- 6개월 더 운영 후 재평가

[시나리오 2: 1년 후에도 정착 X]
- 게임 자체 유지 (작은 업데이트)
- 후속작 개발 중단 / 보류
- 굿즈 / IP 확장 보류
- 개발자 다음 프로젝트 준비

[시나리오 3: 비판 / 컴플라이언스 문제]
- 즉시 사과 + 수정
- 사행성 우려 시 — 게임 디자인 변경 검토
- 한국 게임위 / Apple / Google 가이드 준수

9-2. 출구 전략 (Exit)

인디 게임 출구 옵션:
- 자체 운영 지속 (가장 흔함)
- 퍼블리셔 협업 (인디 → 인디 퍼블리셔)
- IP 매각 (드물지만 가능 — 후속작 미개발 시)
- 오픈소스 (게임 자체 — 장기, 옵션)

K-Sutda 의 경우 자체 운영 지속이 가장 가능.

10. 결론

K-Sutda 의 비전은 단일 게임이 아닌 — 한국 인디 카드 게임의 한 축.

단기 (1년): 다인 플레이 + DLC + 모바일
중기 (2~3년): 후속작 + 글로벌 + 콘솔
장기 (5년+): IP 확장 (웹툰 / 영상 / 보드게임)

성장의 기준:
1. 한국 정체성 강화 (영합 X)
2. 데이터 기반 결정 (감 X)
3. 윤리적 디자인 (사행성 회피)
4. 자생적 커뮤니티 (강요 X)
5. 인디 정체성 유지 (대기업 / Pay-to-Win 회피)
