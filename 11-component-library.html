K-Sutda — 컴포넌트 라이브러리

재사용 UI 컴포넌트의 시각 + 상태 + 사양 + 코드 매핑. Figma 컴포넌트 등록 + Godot 구현 인터페이스.

0. 컴포넌트 분류

0-1. 카테고리

카테고리 컴포넌트
Atoms (원자) Label, Button, Input, Slider, Icon, Divider
Molecules (분자) Card, MapaSlot, StatCard, ScoreBadge, ToastMessage
Organisms (유기체) BettingPanel, PrestigeBar, DeckTrackerBar, ResultPanel
Templates (템플릿) Board layout, Market layout, MainMenu layout

본 문서는 Atoms + Molecules + Organisms 중심.

0-2. 컴포넌트 명명 규칙

  • PascalCase: BettingPanel, MapaSlot, CardVisual
  • 변형: Button.Primary, Button.Danger, Toast.Bluff
  • 상태: Button[hover], Card[face_up]

2. Molecules — 분자 컴포넌트

2-1. Card (CardVisual)

현재 코드: scripts/game/Card.gd

속성 사양
사이즈 220 × 320 px
face_up 한지 베이지 배경 + 월/type 일러스트 (디자이너 작업)
face_down 짙은 녹색 + 격자 패턴 + 중앙 마크
광패 강조 4px 금색 보더 (자동 감지)
정보 표시 좌상단 "%d월" (12px 흰색) + 우하단 type 한글 (14px 흰색)
모션 M-01 fade-in / M-02 flip / M-03 슬라이드인

상태:

  • default — 정적
  • clickable — mulligan 가능 시 hover 글로우
  • partial_reveal — Sprint 3.8 info_reveal 효과 (월만 / 월+type 노출)

Godot Variants (현재 코드):

  • set_card(data, face_up) — 카드 설정
  • flip(face_up) — 뒤집기 애니메이션
  • set_partial_reveal(level) — 부분 노출

2-2. MapaSlot

현재 코드: scripts/game/MapaSlot.gd

상태 시각
empty 빈 슬롯, 회색 보더 점선
equipped 마패 카드 미니뷰 + 희귀도 색 보더
equipped[selected] 추가 글로우 (swap 모드)
equipped[triggered] M-13 펄스 (효과 발동)
희귀도 보더 색
common — 회색 #8C8C8C
uncommon — 청 #338CD9
rare — 자주 #C74DD9
legendary — 주황 #F39C2E + 글로우

사이즈: 100 × 140 px (Board 슬롯) / 200 × 280 px (Market 진열)

8. 컴포넌트 작업 우선순위

Tier 1 — 베타 직전 (디자이너 시안 필수)

  • Button (Primary/Medium/all states)
  • Card (220×320 — 화투 40장 + 뒷면 — 05번 가이드 참조)
  • MapaSlot (4 rarities)
  • BettingPanel (1차/2차)
  • StatCard (DailyActivity)
  • PrestigeBar (3 states)

Tier 2 — 베타 후

  • ToastMessage (5 variants) + Toast 시스템
  • Modal 시스템 (Soul 카드 발견 등)
  • 추가 Icon SVG (이모지 → 정식 아이콘)
  • Settings 페이지 컴포넌트 (Slider, Toggle)

Tier 3 — 폴리시

  • 모든 Button variants × all states (Hover/Pressed/Disabled 시안)
  • Loading spinner / progress indicator
  • Tooltip 컴포넌트 (Sprint 3.8 일부 구현)
