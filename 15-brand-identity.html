K-Sutda — 브랜드 아이덴티티 (Brand Identity)

로고 사용 규정 / 컬러 변형 / 잘못된 사용 예시 / 톤 앤 매너 / 슬로건 변형. 모든 마케팅 / 디자인 결정의 기준.

0. 브랜드 핵심

0-1. 브랜드 한 줄

K-Sutda — 한국 화투의 무게감을 가진 로그라이크 카드 게임

0-2. 브랜드 가치 (Values)

가치 의미
진중함 가벼운 카지노 게임이 아닌, 인생의 한 판
한국적 정체성 화투·노름판·전통 — 글로벌 시장에서 한국 정체성 강조
전략의 깊이 운만이 아닌 빌딩·블러프·메모리
윤리적 디자인 도박 미화 ❌, 가상 명성치만, 사행성 회피

0-3. 톤 앤 매너 (Tone & Manner)

항목 톤
무게감 진중함 (가벼움 ❌)
속도 결단적, 짧음 (장황 ❌)
정서 한국적 (코스모폴리탄 ❌, 일본 ❌)
표현 한국 전통 어휘 + 도박 슬랭
유머 절제, 격조 있는 미소 (코미디 ❌)

1. 로고 시스템

1-1. 메인 로고

┌──────────────────────────┐
│                          │
│   K-Sutda                │  ← 영문 한자 톤
│                          │

조형 원칙:

  • 한국 전통 글씨체 (명조 또는 궁체 톤) + 약간 모던
  • 굵직한 라인 — 도박판의 무게감
  • "K-" 와 "Sutda" 사이 약간의 시각적 휴식
  • 색: 금색 (#F3C72E) 메인, 흰색 / 검은색 단색 변형

1-2. 변형 (Variants)

변형 사용처 색
메인 컬러 Steam capsule / 메인 마케팅 금색
흰색 단색 어두운 배경 흰색
검은색 단색 밝은 배경 (한지 베이지 등) 짙은 먹
모노톤 (도장 스타일) 작은 아이콘 / 모바일 단색
한자 변형 한국 코어 / 옵션 "韓鬪" 또는 "韓섯다"

1-3. 가로 vs 세로

  • 가로 (Horizontal): 메인 — Steam capsule / 헤더
  • 세로 / 정사각 (Stacked): 모바일 아이콘 / SNS 프로필
    • "K-" 위, "Sutda" 아래 stack
    • 또는 한자 1자 (도장 형태) — 모바일 아이콘

규칙: 로고 주변에 최소 로고 높이의 0.5배 여백 — 다른 텍스트 / 이미지 침범 ❌.

1-5. 최소 사이즈

변형 최소 폭
가로 로고 120 px (작아지면 가독성 ↓)
세로 / 정사각 48 × 48 px (모바일 아이콘)
모노톤 도장 32 × 32 px

2. 잘못된 로고 사용 (Don'ts)

2-1. 시각적 변형 ❌

❌ 늘리기 (가로 / 세로 비율 깨기)
❌ 회전 (90도 / 사선)
❌ 그림자 / outline 임의 추가
❌ 외부 색상 (브랜드 컬러 외)
❌ 배경 색과 명도 대비 부족 (가독성 ↓)
❌ 그라데이션 (브랜드 톤 안 맞음 — 단색만)
❌ 효과 (네온 글로우 / 형광)

2-2. 컨텍스트 ❌

❌ 카지노 색 (적색 카펫 / 화려한 배경)
❌ 도박 미화 이미지 (돈다발 / 잭팟)
❌ 일본 색감 (벚꽃 분홍 우키요에 톤)
❌ 발라트로 직접 인용 / 비교
❌ 다른 브랜드 (한게임 / 손오공 등) 로고 옆 배치

색 HEX 사용

3-3. 컬러 사용 비율 (60-30-10 규칙)

60% — Primary 배경 (먹 #0D2619)
30% — Secondary / 카드 (한지 #F5EBD1)
10% — Accent / 강조 (금 #F3C72E / 주홍 #D93333)

→ 한국 도박판 무게감 유지 (밝은 색 적게).

3-4. 컬러 사용 금지

금지 이유
❌ 형광 컬러 (네온 핑크 / 라임 그린) 브랜드 톤 안 맞음
❌ 카지노 적색 (밝은 빨강) 도박 미화
❌ 일본 사쿠라 분홍 (강조) 일본 톤
❌ 보라색 메인 한국 도박판 톤 안 맞음

4. 타이포그래피 (Typography)

4-1. 폰트 시스템 (✅ 2026-05-16 미팅 확정)

종류 한글 폰트 (✅ 확정) 영문 fallback 크기
Display (큰 로고) Noto Serif KR Bold Noto Serif Bold 96~120px
Heading Noto Serif KR Bold Inter Bold 24~32px
Body Pretendard Regular Inter Regular 16px
Caption Pretendard Light Inter Light 12px
Score Pretendard Black Inter Black 32~96px

→ Title 계열 = Noto Serif KR 명조 (한국 전통) + Body 계열 = Pretendard (모던 가독) 융합.

4-2. 폰트 분위기

  • 명조 (Serif) = 진중함, 격조 — Title / Logo / 헤더
  • 고딕 (Sans-serif) = 가독성, 모던 — 본문 / UI
  • 혼합 사용 권장 — 한국 전통 + 모던 융합

4-3. 폰트 라이선스

  • 모두 무료 상업 라이센스 폰트 사용
  • 게임 배포 시 라이선스 동봉 필수
  • 브랜드 가이드 PDF 에 폰트 사용 명시

6. 이미지 / 일러스트 가이드

6-1. 메인 키 비주얼

[07-marketing-assets §5 참조]

핵심: 무명 (플레이어) 가 광땡 패 (3월광 + 8월광) 들고 있는 모습

6-2. 일러스트 톤

  • 한국 민화 + 약간 모던
  • 색감 차분 (브랜드 컬러 60-30-10)
  • 라인: 굵은 먹선 + 부드러운 채색
  • 사실주의 / 카툰 / 픽셀 ❌ → 한국 일러스트 톤 ✅

6-3. 사진 사용 (선택)

  • 게임 자체 스크린샷 우선
  • 실제 화투 사진 (소품) — 카테고리 페이지 옵션
  • 실제 노름판 사진 ❌ (현실 도박 인식 위험)

7. 보이스 (Voice) — 브랜드의 인격

7-1. 브랜드가 "사람"이라면?

  • 나이: 50~60대 남성
  • 직업: 한 때 도박판의 마패, 지금은 절제 + 관찰자
  • 말투: 짧고 진중. 가끔 격조 있는 미소.
  • 취향: 한국 전통 (서예 / 차) + 모던 (필요한 만큼 기술)
  • 태도: 한국 도박판의 영광과 그림자 모두 인정

7-2. 브랜드 보이스 (SNS / 게시)

상황 톤
신규 출시 알림 "오랜만에... 새 한 판이 시작됩니다." (격조)
업데이트 패치 "...몇 가지 다듬었소. (변경사항)" (절제)
사용자 광짤 공유 "...광이 보였군." (간결한 인정)
일반 안내 "내일 점검 있겠습니다." (단정)

→ "안녕하세요 K-Sutda 입니다! 🎉" 같은 가벼운 톤 ❌.

7-3. 영문 보이스

  • 한국 보이스를 직역 ❌
  • 영문권 정서 + 한국 정체성 균형
  • 예: "K-Sutda · A new hand begins." (단정, 약간 mythic)

8. 사용 사례 — Do & Don't

8-1. Steam Capsule

✅ Do
┌──────────────────────────┐
│        K-Sutda           │ ← 금색 로고 메인
│   ┌──┐ ┌──┐              │
│   │광│ │땡│              │
│   └──┘ └──┘              │
│   "광 한 번 보는 데       │
│    인생을 건다."          │
└──────────────────────────┘
배경: 먹 (#0D2619)

❌ Don't
- 카드를 화려하게 흩어 놓음 (카지노 톤)
- "Win Big!" 같은 영문 ❌
- 그라데이션 배경 (잭팟 톤)
- 일본 사쿠라 분홍 강조

8-2. SNS 게시

✅ Do
[금색 로고 + 짧은 카피]
"오늘의 카드: 8월 공산광"

❌ Don't
[일러스트 + 긴 설명문 + 이모지 다수]
"😍 와! 오늘 진짜 운 좋아요! ✨🎉 광땡 떴어요!! 🎴"

8-3. 모바일 앱 아이콘

✅ Do
- 단순 + 강한 임팩트 (광패 1장 + "K")
- 배경 짙은 먹
- 모서리 자동 라운드 (iOS/Android 시스템)

❌ Don't
- 너무 자세한 일러스트 (작은 사이즈 인지 ↓)
- 텍스트 위주 (아이콘 = 시각)
- 카드 더미 / 돈 등

9. 디자이너 / 마케터 체크리스트

새 마케팅 / 디자인 자료 만들 때:

  • 로고 사용: 변형 / 클리어 스페이스 / 최소 사이즈 준수?
  • 컬러: 브랜드 Palette 만 (60-30-10 비율)?
  • 폰트: 한글 폰트 + 영문 fallback 정의?
  • 카피 톤: 진중 / 결단 / 한국적?
  • 도박 미화 ❌, 한국 정체성 ✅?
  • 일본 톤 / 카지노 톤 회피?
  • 사진 / 일러스트 톤 일관?
  • 한↔영 양쪽 톤 어울림?
  • 작은 사이즈에서도 인지 가능 (모바일 아이콘)?

11. 브랜드 거버넌스

11-1. 변경 권한

  • 로고 / 슬로건 / 톤 = 게임 개발자 + 디자이너 합의
  • 외부 협업자 (마케팅 / 인플루언서) 는 본 가이드 준수

11-2. 라이선스

  • 로고 / 슬로건 / 브랜드 자산 = 게임 측 저작권
  • 외부 사용 (인플루언서 영상 등) 시 동의 필요
  • 라이선스 PDF 또는 BrandKit 다운로드 페이지 (옵션)

11-3. 버전 관리

  • 모든 브랜드 자산 git 추적
  • 변경 시 commit message 에 "Brand: ..." 태그
  • 분기별 브랜드 사용 모니터링 (SNS / 외부)

12. 글로벌 확장 시

12-1. 한국 → 영어권

  • 슬로건 / 카피 영문 버전 (위 §5 참조)
  • "Sutda / Mapa / Gwangddaeng" 한국 고유 명사 보존
  • 영문 사용자 첫 접속 시 짧은 설명 ("Sutda — a traditional Korean card game")

12-2. 한국 → 일본권

  • 일본 화투 (花札) 와 혼동 피하기 — "Korean Sutda" 명시
  • 슬로건 일본어 — 격조 있는 톤 (한국과 비슷한 무게감)
  • 캐릭터 이름 / 한자 — 일본 한자 표기 변형 (필요 시)

12-3. 한국 → 중화권

  • "韓鬪" 한자 로고 적극 활용 (중화권 친숙)
  • 도박 콘텐츠 검열 주의 — 게임 내 disclaimer 강화

13. 결론

K-Sutda 브랜드는 한국 도박판의 무게감과
글로벌 인디 게임의 매력 사이의 균형입니다.

모든 결정은 다음 질문으로 검증:
1. "이건 한국적인가?"
2. "이건 진중한가?"
3. "이건 도박 미화 ❌, 전략 게임 ✅?"
4. "이건 글로벌 사용자도 이해 가능한가?"

답이 "한국적, 진중, 게임, 글로벌" 이면 진행.
