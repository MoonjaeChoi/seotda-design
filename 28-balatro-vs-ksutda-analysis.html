28. 발라트로 vs K-Sutda 진행 방식 비교 분석

작성일: 2026-05-19
목적: 발라트로(Balatro)와 K-Sutda의 게임 진행 방식을 구조적으로 비교 분석해 개발 방향 및 차별화 전략을 정리한다.

1. 핵심 구조 비교

항목 발라트로 K-Sutda
기반 카드게임 포커 (52장 표준) 섯다 (화투 40장)
런 구조 8 Ante × 3 Blind = 24라운드 8 단계 × 3판 = 24판
점수 공식 Chips × Mult (base_score + flat_bonus) × mult_chain
명성치 환산 없음 (Chips 직접 비교) final_score ÷ 20 = 명성치 획득
조커 슬롯 5개 (+Negative 에디션으로 확장) 3개 (+Reverse 에디션 stub)
조커 풀 150종 65종
AI 상대 없음 (Blind vs 목표 점수) 있음 — AI도 마패패 보유
멀티플레이 없음 Phase 4 로드맵
정보 비대칭 없음 핵심 메카닉 (1장 후 베팅)

2. 판 단위 흐름 비교

발라트로 판 흐름

Hand 선택 → 점수 계산 (Chips × Mult) → 목표 달성 여부
  └─ 조커 좌→우 발동 → 카드 채점 → 최종 점수
  └─ 전략: 어떤 Hand를 내느냐

K-Sutda 판 흐름

카드 1장 → [1차 베팅: call / raise / dia]
  → 카드 1장 → [Mulligan?] → [2차 베팅: call / raise / all-in / fold]
    → 마패패 발동 (T1~T6) → 패 공개 → 승패 판정
  └─ 전략: 베팅 심리전 + 블러프 + 족보 빌드

K-Sutda는 같은 족보라도 베팅 결정이 결과를 바꿉니다.
발라트로는 패 선택 후 결과가 확정적입니다.

3. 조커(마패패) 시스템 비교

항목 발라트로 K-Sutda
발동 시점 단순 (패 제출 시) T1~T6 6단계 (앤티/패확인/베팅/공개전/승패/판종료)
시너지 색·suit·rank·조건부 다수 ❌ 없음 (격차)
에디션 Foil / Holo / Polychrome / Negative Silver / Gold / Rainbow / Reverse
AI 조커 없음 (AI 없음) 있음 — 단계 5~8 보스 전용 마패 6종
카드 내재 효과 없음 있음 — 화투 40장 각각 효과
발동 시각화 숫자 카운트업 애니메이션 점수 폭발 UX (Phase A~F 완료)

마패패 발동 타이밍 (K-Sutda 6단계)

타이밍 발동 조건 효과 유형
T1 — 앤티 시 판 시작 economy, mult_accumulate
T2 — 패 확인 직후 두 번째 카드 수령 후 info_reveal, deck_manipulation
T3 — 베팅 행동 시 raise / all-in 발생 mult_immediate, bluff_synergy
T4 — 패 공개 직전 showdown 직전 info_reveal_and_mult
T5 — 승패 판정 시 결과 확정 psychological_pressure
T6 — 판 종료 후 결과 이후 mult_accumulate (누적 스케일링)

5. 단계 진행 구조 비교

발라트로 Ante 구조

Ante 1~8 (각 3 Blind)
  Small Blind → Big Blind → Boss Blind
  Boss Blind = 특수 제약 조건 (특정 족보 점수 X 등)

K-Sutda 단계 구조

단계 1~8 (각 3판)
  소판 → 대판 → 최종판 (보스 AI 대결)
  보스 = AI가 단계별 특수 마패패 보유

단계별 명성치 목표

단계 목표 명성치 AI 패턴 특징
1 250 BASIC 튜토리얼
2 252 BASIC Pure Score 학습
3 263 AGGRESSIVE 압박 시작
4 700 AGGRESSIVE 본격 빌드 필요
5 1,100 DEFENSIVE 부처의 눈 (광 감지)
6 1,800 INFORMED 올가미의 미소 (raise 함정)
7 14,000 PSYCHOLOGICAL 각시탈 (가짜 압박)
8 35,000 COMPLEX 전설의 손 + 검의 도

7. K-Sutda만의 강점 (발라트로에 없는 것)

7-1. 심리전 베팅 구조

발라트로는 "맞추기 게임" — 어떤 패를 내서 점수를 쌓느냐.
K-Sutda는 "상대 읽기 게임" — 어떻게 베팅해서 상대를 제압하느냐.

2단계 베팅 (1차 call/raise/dia → 2차 call/raise/all-in/fold)은 정통 한게임 섯다와 동일한 구조입니다.

7-2. 정보 비대칭 메카닉

  • 카드 1장만 보고 1차 베팅 결정
  • Mulligan 비용 (8명성치) 으로 불리한 카드 교체 가능
  • DeckTracker로 잔여 카드 카운팅

7-3. AI vs Player 마패 대결

보스 단계 5~8에서 AI도 특수 마패패 보유:

단계 보스 마패 효과
5 부처의 눈 플레이어 광패 감지 → score_factor
6 올가미의 미소 AI raise 시 mult 누적 (함정, 첫 만남엔 숨김)
7 각시탈의 미소 fake pressure toast (실제 영향 없음, 심리전)
8 전설의 손 + 검의 도 고점수 + 공격적 베팅

7-4. 블러프 보상 시스템

약패(망통/끗 1~3)로 raise → AI가 dia/fold → bluff 성공
→ bluff_synergy 마패패 발동 → 추가 점수/배수

발라트로에는 이 구조가 없습니다.

7-5. 화투 40장 내재 효과

발라트로는 카드(포커 핸드)에 효과가 없습니다.
K-Sutda는 모든 화투 패에 타이밍별 효과 내재 — 마패패와 시너지 가능.

8. K-Sutda의 현재 격차 (솔직한 평가)

격차 항목 발라트로 K-Sutda 현재 우선순위
조커 종수 150종 65종 Phase 2
조커 시너지 풍부 ❌ 없음 즉시

난이도 계층 Stake 8단계 ❌ 없음 Phase 3

9. 즉시 개선 시 기대 효과

현재 K-Sutda 완성도를 가장 빠르게 높일 수 있는 3가지:

우선순위 1 — stub 효과 구현 (info_reveal 9개 + deck_manipulation 16개)

현재 25개 마패패 효과가 등록만 되고 실제로 아무것도 안 함.
→ 구현 완료 시 전략 선택지 약 2배 증가.

우선순위 2 — 마패 시너지 1~2종 도입

단일 카드 강한 것 픽이 항상 정답인 구조 탈피.
예) "mult_immediate 2개 이상 시 추가 ×1.5"
→ 빌드 다양성 도입.

우선순위 3 — Voucher/소모 아이템 1~2종

경제 구조 단순화 해소.
예) "이번 런 모든 mult_accumulate 효과 ×1.5" 영구 업그레이드.

10. 결론

"발라트로가 포커로 한 것을 섯다로 한다."
단, 기존 섯다 게임의 도박 구조를 완전히 탈피하고 로그라이크 전략 게임으로 재탄생시킨다.

K-Sutda는 발라트로 대비 심리전·정보 비대칭·AI 대결 세 가지 축에서 명확한 차별화를 보유합니다.
stub 효과 구현과 시너지 1~2종만 추가되면 "섯다 심리전 특화 로그라이크"로 충분한 독자 포지션을 확보할 수 있습니다.
