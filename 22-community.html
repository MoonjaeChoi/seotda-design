0. 커뮤니티 전략

0-1. 왜 커뮤니티인가

  • 솔로 개발자 의 큰 약점 = 사용자 직접 접점 부족
  • 발라트로 LocalThunk 사례 — Discord 적극 활용 → 커뮤니티 자산
  • K-Sutda 특성 — 결정성 (시드 공유), 빌드 토론, 광짤 공유 — 커뮤니티 친화

0-2. 목표

단계 목표
베타 코어 100명 (피드백 / 버그)
정식 출시 활성 500~1,000명
1년차 5,000+ (글로벌 포함)
장기 자생적 커뮤니티 — 사용자 끼리 토론 / 이벤트

0-3. 톤

  • ✅ 친근 + 격조 (브랜드 보이스 — 15-brand-identity §7)
  • ✅ 한국 + 글로벌 양쪽 채널 분리
  • ❌ 과한 가벼움 / 이모지 폭발
  • ❌ 마케팅 일색 (사용자 컨텐츠 우선)

9. 한국 vs 글로벌 커뮤니티 차이

9-1. 한국 사용자

  • 한국 화투 친숙 — 룰 설명 ↓
  • 도박 게임 경험 다수 — 빠른 적응
  • 커뮤니티 활성도 ↑ (특히 디스코드 / 카페)
  • 인플루언서 영향 큼

9-2. 글로벌 사용자

  • Korean game 호기심 — 룰 설명 ↑
  • 발라트로 비교 자연스러움
  • Reddit / Twitter 중심
  • 자막 / 영문 콘텐츠 필요

9-3. 채널 분리 이유

  • 한국 채널 — 한국어 자유, 한국 도박 슬랭
  • 글로벌 채널 — 영어 표준
  • 양쪽 다 활성화 (개발자 부담 ↑ 하지만 가치 큼)

10. 위기 대응

10-1. 부정적 리뷰 / 트렌드

  • 즉시 무시 ❌
  • 정중히 답변 + 개선 약속
  • 큰 이슈는 #공지 에 사과 / 설명
  • 1주일 이내 hotfix 또는 변경

10-2. 사기 / 가짜 베타 키 배포

  • 공식 베타 키 = 개발자 직접 배포
  • 비공식 키 발견 시 즉시 차단 + 공지

10-3. 모더레이션 부담
