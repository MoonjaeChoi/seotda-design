[게임 소개]

K-Sutda 는 한국 전통 카드 게임 섯다에 발라트로 (Balatro) 식
로그라이크 빌딩을 접목한 싱글플레이어 전략 카드 게임이다.

발라트로의 빌딩 / 시너지 / 보스 시스템에 한국 화투의 심리전 / 블러프 /
2단계 베팅을 결합. 65종 마패 카드로 자신만의 빌드를 만들어
8단계의 한국 도박판을 클리어하는 것이 목표.

[핵심 차별화]

1. 한국 정통 2단계 베팅: 첫 카드 1장 보고 1차 베팅 →
   2번째 카드 받은 후 2차 베팅. 결단의 깊이.

2. 67종 마패 카드: 광패 시너지 / 블러프 보너스 / 연승 누적 등
   복잡한 빌드 시스템.

3. 8단계 한국 보스: 동네 형 박달삼부터 전설의 마패 남궁천까지.
   각 보스의 독특한 AI 패턴과 백스토리.

4. 결정성 (Deterministic): 같은 시드 = 같은 런. 친구와 도전 가능.

5. 인앱결제 / 광고 없음: 정액 구매. 사행성 모사 회피.

[게임 시스템]

- 8 단계 × 3 라운드 (1 단계 = 소판 / 대판 / 보스 라운드)
- 매 라운드: 1차 베팅 (1장 보고) → 2차 베팅 (2장 완성 후)
- 명성치 (가상 점수) -150 도달 시 게임오버
- 마패 슬롯 3개 — 시장에서 구매 / 슬롯 교환
- 마지막 단계 8 클리어 → "전설" 칭호

[ABOUT THE GAME]

K-Sutda is a single-player roguelike deckbuilder inspired by
Korean Sutda — the traditional two-card mind game — and built
with the building blocks of Balatro.

Combine Balatro's signature building / synergy / boss system with
Sutda's mind game / bluffing / two-stage betting. Build your way
through 67 Mapa cards and 8 stages of Korean gamblers to claim
the title of "Legend".

[KEY FEATURES]

1. Two-Stage Betting (Korean tradition): See one card and decide.
   See the second and decide again. Depth in every hand.

2. 67 Mapa Cards: Gwang synergy, bluff bonuses, win streak
   compounds. Build your way to victory.

3. 8 Korean Bosses: From a neighborhood gambler "Dalsamee" to the
   legendary master "Nam-gung Cheon". Each with unique AI and
   backstory.

4. Deterministic: Same seed = same run. Challenge friends.

5. No IAP / No Ads: Premium one-time purchase. Ethical design,
   no gambling glorification.

[GAME SYSTEM]

- 8 stages × 3 rounds (each stage = small / large / boss round)
- Per round: 1st bet (see 1 card) → 2nd bet (after 2 cards)
- Game Over at prestige ≤ -150
- 3 Mapa slots — buy from market / swap freely
- Beat stage 8 → earn "Legend" title

7. 개발자 인터뷰 Q&A (예시)

Q1. 왜 한국 섯다인가요?

한국 섯다는 단순히 도박 게임이 아닌, 한국 도박판의 정서와
한국 사회의 한 단면을 담는 카드 게임입니다.

발라트로 같은 인디 카드 게임이 글로벌하게 사랑받는 시대에,
한국 전통 카드 게임의 무게감을 글로벌 사용자에게 전하고 싶었습니다.

또한 한국 화투는 한국 가정 / 마을 모임의 추억과
어두운 도박 문화의 양면을 모두 담고 있어, 게임의 톤으로
풍부한 소재였습니다.

Q2. 발라트로와의 차이점은?

발라트로는 포커 + 조커 시스템의 빌딩 즐거움이 메인입니다.
K-Sutda 는:

1. 한국 정통 2단계 베팅 — 카드 1장 보고 1차, 2장 완성 후 2차.
   결정의 깊이가 2배.
2. 한국 화투의 시각·정서 — 차분하고 진중한 한국 도박판 톤.
3. 8 단계 보스 — 각 캐릭터의 한국적 백스토리 + 패턴.
4. 블러프 시스템 — 약한 패로 raise → 상대 다이 유도.

발라트로의 빌딩 매력 + 한국 섯다의 심리전 = K-Sutda.

Q3. 사행성 모사 문제는 어떻게 회피했나요?

한국 게임산업법 제33조의3 및 글로벌 게임 윤리에 따라:

1. 명성치 = 게임 내 가상 점수 (실제 환금성 X)
2. 모든 화면에 disclaimer ("게임 내 가상 점수입니다")
3. 도박 미화 어휘 회피 ("이긴다 / 대박" X, "광 보임 / 결단" O)
4. 화려한 카지노 톤 회피 (한국 전통 도박판 톤)
5. 광고 / 인앱결제 / 가챠 없음

도박 게임이 아닌, 한국 전통 카드 게임을 모티프로 한
전략 카드 게임입니다.

Q4. 가장 자랑하는 시스템은?

2단계 베팅 시스템과 블러프 보상입니다.

한 판이 1차 + 2차 두 번의 결정으로 진행되어, 결정의 깊이가
2배로 늘어납니다. 약한 패로 1차 raise → 상대 dia 유도 →
"블러프 성공" 보상까지 — 단순한 운 게임이 아닌, 진짜 심리전을
구현하고자 했습니다.

이건 한국 한게임 섯다의 정통 흐름이기도 합니다.

Q5. 다인 플레이 계획은?

출시 6개월 후 다인 플레이 모드 (4~6인) 추가 예정입니다.

- 로컬 다인 (같은 PC) 우선
- 온라인 다인 (P2P 또는 매칭) — 시점 검토 중
- 한국 + 글로벌 매칭 (인프라 결정)

다인 플레이는 K-Sutda 의 본질 — 한국 섯다는 본래
다인 게임이고, 정보 비대칭과 메모리 게임이 깊이를
크게 더할 것입니다.

9. PR 캠페인 일정

9-1. 출시 전 (6주 ~ 1주 전)

[6주 전] Press Kit 완성 + Steam 페이지 공개
        → Steam Page Launch + 위시리스트 캠페인

[4주 전] 메인 트레일러 공개
        → YouTube + Twitter + Discord

[3주 전] 인플루언서 키 배포 (50명+)
        → 영상 / 스트림 일정 조율

[2주 전] 한국 미디어 보도자료 (인벤 / 디스이즈게임)
        → 기사 게재 요청

[1주 전] 글로벌 미디어 보도자료 (PC Gamer / IGN 등)
        → 기사 게재 요청

[3일 전] Last Push — Discord / SNS / 인플루언서 한 번 더

9-2. 출시 당일

[D-Day]
- 한국 시간 오전 10시 출시 (글로벌 친화 시각)
- 트위터 / Discord / Steam 공지
- 한국 + 글로벌 미디어 기사 동시 배포
- 인플루언서 영상 동시 공개
- 출시 기념 트레일러 (옵션)

9-3. 출시 후 (1주 ~ 1개월)

[D+1주] 첫 주 리뷰 모니터링 + 응답
       → 한국 / 글로벌 리뷰 모두

[D+2주] 첫 패치 (밸런스 / 버그)
       → 패치 노트 + SNS 공지

[D+1개월] 한 달 결산 + 다음 계획
         → Discord AMA + 인터뷰

12. 라이선스 / 사용권

12-1. 미디어 자유 사용

[K-Sutda 미디어 자유 사용 정책]

기자 / 리뷰어 / 인플루언서 / 영상 작가는 다음 자료를
별도 허락 없이 자유롭게 사용 가능:

- 트레일러 (전체 또는 부분)
- 스크린샷 / 로고 / 키 비주얼
- 보도자료 텍스트 (인용)
- 게임플레이 영상 (자기 채널 영상 제작)
- Steam 페이지 정보

조건:
- 출처 표기 ("K-Sutda" 또는 게임 페이지 링크)
- 의도된 무단 변형 ❌ (예: 로고 변형 / 캐릭터 변형)
- 광고 / 상업 사용 시 사전 컨택 (옵션)

12-2. 인플루언서 / 스트리머

  • 게임 영상 / 스트림 = 자유 (모네타이즈 가능)
  • 사전 동의 / 키 배포 (22-community)
  • 출시 전 1주 freeze (스포일러 X)

12-3. 미디어 자료 갱신

  • 분기별 미디어 키트 갱신 (큰 업데이트 시)
  • 새 스크린샷 / 트레일러 추가
  • 보도자료 갱신 (새 콘텐츠 / 마일스톤)
