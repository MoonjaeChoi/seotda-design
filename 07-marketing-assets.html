1-4. 스토어 페이지 텍스트

About This Game (한국어):

한국 전통 카드 게임 섯다에 발라트로식 로그라이크 빌딩을 접목했습니다.

• 2단계 베팅 시스템 — 한 장 보고 베팅, 두 장 완성 후 다시 베팅
• 67종 마패 카드로 자신만의 빌드 — 광패·블러프·연승 시너지
• 8단계 보스 — 동네 형부터 전설의 마패까지
• 시드 기반 결정성 — 같은 시드 = 같은 런
• 100% 싱글플레이, 인앱결제 / 광고 없음

※ 본 게임의 모든 점수는 게임 내 가상 단위입니다. 실제 환금성 없음.

About This Game (English):

A roguelike deckbuilder inspired by Korean Sutda — the traditional
two-card mind game.

• Two-stage betting — see one card, decide. See the second, decide again.
• Build with 67 Mapa cards — gwang synergy, bluff bonuses, win streaks
• 8 stage bosses — from a neighborhood gambler to the legendary master
• Seed-based determinism — same seed, same run
• 100% single-player, no IAP / ads

※ All in-game scores are virtual. No real-world currency value.

Genre Tags:

  • 한국: 카드, 로그라이크, 덱빌딩, 싱글플레이어, 전략, 인디
  • English: Card Game, Roguelike, Deckbuilding, Single-player, Strategy, Indie, Korean Culture

1:05  플랫폼 로고 + 출시 일자
