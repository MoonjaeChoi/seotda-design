작업 시작 전 반드시 ../docs/01-overview.md (게임 개요) + ../docs/03-game-design.md (게임 시스템) 함께 정독 권장. 본 문서는 디자인 작업의 마스터 안내서 입니다. 상세 에셋 목록은 02-asset-spec.md.

1. 게임 한 줄 정의

"섯다의 심리전 DNA에 로그라이크 빌딩을 접목한 싱글플레이어 전략 카드 게임"

레퍼런스:

  • 발라트로 (Balatro) — 점수 빌딩 / 조커 시스템 / 깔끔한 UI
  • 한게임 섯다 (NHN) — 2단계 베팅 / 다이 메커니즘 / 한국 화투 시각
  • 슬레이 더 스파이어 — 로그라이크 진행 / 단계 구조

우리 게임의 톤:

  • 한국 전통 + 모던 도박판 분위기 (먹색 + 금색 + 한지 베이지)
  • 발라트로 같은 산뜻함 ❌ → 진중한 무게감 + 한국적 정서 ✅
  • 사행성 모사 회피 (도박 미화 ❌, 전략 카드 게임 ✅)

2. 타겟 플레이어

  • 코어: 25~45세 한국 남성, 섯다/화투 문화 이해 + 발라트로/로그라이크 경험
  • 세컨더리: K-문화 관심 글로벌 유저, 전략 카드 게임 팬
  • 디자인 톤은 한국 + 약간 모던 (글로벌 친화) 균형

3. ⚠ 법적 제약 (필수 준수)

../docs/02-legal.md 의 사행성 모사 회피 원칙. 디자인 시각 표현도 영향:

금지 권장 대체
실제 화폐 / 금액 표시 "명성치 (名聲)" / "점수"
카지노 적색 카펫 / 슬롯 휠 한국 전통 도박판 (먹색 + 한지)
잭팟 / 대박 시각 효과 점수 폭발 (금색 글로우)
도박 미화 (담배·술·돈다발) 화투 / 한지 / 붓 / 매화

6. 즉시 작업 우선순위 (베타 직전)

Tier 1 — 핵심 게임플레이 비주얼 (필수)

  1. 화투 40장 일러스트 (현재 절차적 _draw() — Card.gd 참조)

    • 1월~10월 × 4 type (광/띠/열/피)
    • 일관된 일러스트 스타일 (한국 전통 화투 + 약간 모던)
    • 광땡 가능 카드 (1월광 / 3월광 / 8월광) 특수 강조 가능
    • 사양: 220×320px @1x (모바일 ×2 ready)

  2. 카드 뒷면 디자인 (현재: 짙은 녹색 + 격자)

    • 한국 전통 문양 (구름 / 매화 / 학)
    • 광땡 강조 보더 (금색)

  3. Board UI 레이아웃

    • 카드 슬롯 (player 2개 + opponent 2개)
    • BettingPanel (1차 주홍 / 2차 금색 — 기존 컬러 활용)
    • PrestigeLabel + PotLabel 강조
    • 도박판 분위기 — 한지·먹·금 조합

Tier 2 — 마패 카드 (67종)

  1. 마패 카드 일러스트 65~67장 (data/mapa_cards.json 참조)
    • 희귀도별 색 보더 (위 RARITY 색 사용)
    • 캐릭터·도구·동물 모티프 (예: "노름꾼의 직감", "광의 신", "구라쟁이")
    • 사양: 200×280px (Market 진열 + Board 슬롯 양쪽 사용)
    • 5종 Legendary는 특별한 일러스트 (Soul card 발견 시 큰 임팩트)

Tier 3 — UI 아이콘 + 효과

  1. 이모지 → 정식 아이콘 교체

    • 🃏 (블러프) / 🎴 (2차 베팅) / ⚔️ (보스) / 🏆 (승리) / ⚠️ (위험)
    • 32×32 SVG + 색상 variants

  2. 점수 폭발 / 카드 슬라이드인 / 펄스 효과

    • 디자이너가 motion graphic 작업 시 참고용 모션 명세 필요

Tier 4 — 사운드 (별도 외주 또는 라이센스)

  1. SE 8개 — card_flip / card_deal / win / lose / button_click / mapa_trigger / first_dia / bluff_success
  2. BGM 2~3개 — 메뉴 / 게임 / 보스 라운드

7. 디자인 원칙 (브랜드)

Do ✅

  • 한국 전통 화투 색 (먹·금·한지·주홍) 적극 사용
  • 광패의 시각적 무게감 (금색 보더, 글로우)
  • 1차 vs 2차 베팅 색 구분 (주홍 vs 금) — 인지 부담 ↓
  • 폰트 위계 명확 (점수 폭발 96px / 일반 16px)
  • 한국 전통 문양 (구름·매화·학·소나무) 차용

Don't ❌

  • 카지노 적색 카펫 / 슬롯 그래픽
  • 돈다발·동전·잭팟 이미지
  • 발라트로 픽셀 아트 직접 모방 (저작권 + 톤 불일치)
  • 형광 컬러 (브랜드 톤 안 맞음)
  • 일본 화투 그대로 — 한국화 (예: 한국식 색감, 한국 식물 정확성)

8. 작업 자료 제공

디자이너 접근권 권장

  • 이 repo (read-only) 또는 docs/ + assets/ 폴더만 따로
  • Figma / Adobe XD 공유 (디자이너가 만든 경우)
  • Slack / 카톡 채널 (피드백 빠른 응답)

추가 참고 자료

  • 한국 화투 시각 레퍼런스: 손오공게임 화투 / 한게임 섯다 카드 디자인
  • 발라트로 UI 톤: 진중함 + 약간 레트로 (https://www.playbalatro.com — 참고만)
  • 한국 전통 문양: 단청, 책가도, 화조도

파일 전달 형식

  • 일러스트: PNG (투명 배경) + 원본 (PSD/AI/SVG)
  • 모션: After Effects 프로젝트 또는 .json (Lottie)
  • 사운드: .ogg (Vorbis, mono SE / stereo BGM)
  • 폰트: 라이센스 명시 (.otf/.ttf)
