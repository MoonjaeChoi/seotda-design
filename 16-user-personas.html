K-Sutda — 사용자 페르소나

핵심 사용자 6명 페르소나. 디자이너 / 마케터 / 게임 디자이너 모두 "이 사용자가 좋아할 것"을 기준으로 결정할 때 참조.

0. 페르소나 개요

0-1. 우선순위

페르소나 타입 비중 (추정)
P1: 발라트로 한국 코어 코어 🔴 35%
P2: 화투 친숙 중년 코어 🟡 20%
P3: 글로벌 인디 카드 게이머 세컨더리 🟡 20%

P5: 캐주얼 모바일 확장 🟢 10%

0-2. 페르소나 사용법

  • 디자인 결정 시: "P1 이 이걸 좋아할까?" 질문
  • 마케팅 카피: P1 + P2 한국 코어 우선, 부제는 P3 영문 톤
  • 튜토리얼 깊이: P5 캐주얼 기준 (가장 적은 사전 지식)
  • 접근성 / 모바일: P5 + P4 글로벌 고려

8. 페르소나 활용 — 디자이너 워크플로우

8-1. 새 화면 시안 시

  • 첫 번째 페르소나: P1 (발라트로 한국 코어) — 메인 사용자
  • 검증 페르소나: P5 (캐주얼) — "이해 가능한가?"
  • 글로벌 검증: P3 (인디 카드) — "한국 정체성 + 영문 자연?"

8-2. 마케팅 카피

  • 한국 메인: P1 + P2 우선 — "광 한 번 보는 데 인생을 건다"
  • 글로벌: P3 + P4 — "One last hand"
  • 인플루언서: P6 — 짧은 클립 + 베타 키 우선

8-3. 접근성 / 모바일

  • P5 가 만족하면 모두 만족
  • 폰트 사이즈 / 터치 타겟 / 튜토리얼 깊이 → P5 기준

8-4. 깊이 / 옵션

  • P1 / P6 만 사용할 옵션 (스피드 모드 / 통계 / 도전과제) 점진 추가
  • P5 가 부담스럽지 않도록 기본은 단순, 옵션으로 깊이

10. 향후 페르소나 추가 (출시 후)

10-1. 잠재 페르소나

  • 한국 노년 (60대+) — 화투 친숙도 ↑, 모바일 사용성 (큰 글씨 옵션)
  • 글로벌 카지노 카드 게이머 — 포커 / 블랙잭 출신, 도박 게임 친숙
  • 한국 모바일 게임 코어 — 리니지 / 메이플 등 MMO 위주, 카드 게임 진입

10-2. 데이터 수집

  • 베타 시작과 함께 사용자 설문 (옵션) — 페르소나 분포 측정
  • 일별 활동 데이터 → 사용자 행동 패턴 분석
  • 분석 후 페르소나 가중치 재조정 (출시 1개월 후)
