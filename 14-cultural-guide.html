K-Sutda — 한국 문화 가이드

화투 / 노름판 / 한국 도박 문화에 대한 디자이너 / 작곡가 / 카피라이터 컬처럴 레퍼런스. 한국 정서 + 시각·청각 비주얼 자료.

0. 왜 컬처럴 가이드인가

K-Sutda 의 핵심 차별화는 한국 도박판의 정서·시각 재현. 디자이너가 일본 화투 / 카지노 / 발라트로 톤을 가져오면 게임 정체성 약화.

본 가이드는 한국 노름판 = 단순 게임이 아닌 한국 사회의 한 단면 임을 디자이너가 이해하도록 함. 시각·음향 결정에 영향.

1. 화투 (花鬪) 문화 배경

1-1. 역사

  • 기원: 일본 하나후다 (花札) 가 19세기 후반 한국에 전파
  • 한국화: 한국 식물·동물·정서로 점진 변형 — 현재 한국 화투는 일본 하나후다와 시각·게임 룰 모두 다름
  • 시기: 일제강점기 (1910~1945) 동안 유행 → 해방 후 한국 토착화 가속
  • 사회적 위치: 도박 도구 + 가족 / 마을 놀이 양면 (설날 친척 모임 화투 등)

1-2. 한국 화투의 정서

  • 한국적: 가족 모임 / 마을 평상 / 작은 판
  • 이면: 도박 중독, 가산 탕진, 가족 갈등 — 한국 영화 "마패" 등에서 묘사
  • 이중성: 우리가 가져갈 톤 — "가벼운 놀이" 가 아닌 "인생을 거는 한 판"

1-3. 일본 화투 vs 한국 화투

항목 일본 (花札) 한국 (화투)
색감 채도 높음, 화려 차분, 짙은 톤
식물 묘사 우키요에 평면, 가는 라인 한국 민화 + 묵직한 라인
게임 룰 코이코이 / 하나후다 점수 섯다 / 고스톱 (한국 고유 변형)
사회 인식 가족 놀이 (정서 가벼움) 도박 어두운 면 강함

→ K-Sutda 는 한국 화투 + 어두운 정서 톤.

2. 섯다 (Sutda) 게임의 본질

2-1. 섯다 = 2장의 심리전

  • 단순성: 룰 단순 (카드 2장 + 족보 + 베팅)
  • 복잡성: 심리전 + 베팅 압박 + 블러프 + 상대 패턴 학습
  • 한 판의 무게: 한 판 = 짧은 시간, 큰 베팅 → 인생의 압축

2-2. 한국 영화 / 드라마의 섯다

작품 섯다 묘사
마패 (2006) 한국 도박판 본격 묘사. 캐릭터 / 룰 / 정서 레퍼런스 풍부
마패: 신의 손 (2014) 후속 — 신예 마패 시점
꽁트 (드라마) 평범한 가족 화투, 코미디 톤 (우리 톤 아님)
광해 (2012) 조선시대 도박판 (역사적)

디자이너 시청 권장: "마패" (2006) — 단계 보스 캐릭터, 도박판 분위기, 한국 도박 슬랭 모두 풍부.

3. 한국 노름판 비주얼 레퍼런스

3-1. 공간

공간 분위기 비주얼
동네 평상 가벼움, 시골 나무 평상, 막걸리, 담뱃대, 가벼운 화투
시장 좌판 활기, 도시 시장 등불, 천 더미, 야간
지방 도박장 어두움, 본격 싸구려 네온, 위스키, 돈 무더기
사찰 신비, 명상 향, 염주, 노을 진 절
고급 다실 격조, 최고 한지 매트, 정갈한 다구, 백자, 묵향

각 단계 보스 (03-characters) 배경에 매칭.

3-2. 소품

  • 화투 카드 — 두꺼운 종이, 모서리 약간 닳은 느낌
  • 막걸리 / 소주 / 위스키 — 단계별 차등
  • 담뱃대 (한국 전통) / 담배 — 한국 도박판 단골
  • 돈 (지폐) → 명성치 (가상) — 우리는 가상 화폐만
  • 다구 (茶具) — 고급 단계
  • 부적 / 염주 — 사찰 / 운 빌기

3-3. 의상

단계 의상
1 (동네 형) 흰 셔츠 + 멜빵바지 / 평상복
2 (시장 상인) 앞치마 / 시장 옷차림
3 (신참 마패) 양복 셔츠 (풀어헤침)
4 (군 출신) 카키 자켓 / 군 출신 다부진
5 (스님) 회색 승복
6 (미녀 사기꾼) 짙은 한복 / 정장 한복
7 (탈춤 영감) 색동 한복 + 양반탈
8 (전설) 검은색 정장 한복

→ 단계 진행 = 의상 격조 증가 (평범 → 신비 → 전설).

3-4. 색감

  • 짙은 먹색 / 짙은 녹 (도박판 베이스)
  • 한지 베이지 (카드 / 매트)
  • 금색 액센트 (광패 / 강조)
  • 주홍 (위험 / 띠)
  • 청록 (스님 / 성공)
  • 어두운 자주 / 진남 (밤)

→ 모두 Palette.gd 에 반영됨.

4. 한국 도박판 매너 / 룰

4-1. 매너 (한국 정서)

  • 선 (先) 결정 — 첫 베팅 누가 먼저 (보통 직전 승자)
  • 다이의 무게 — "다이" 는 단순 "포기" 가 아닌 "이번 판은 보내겠소" 의 격조
  • 승자의 침묵 — 큰 승리에도 말 적게 ("운이 좋군" 정도)
  • 패자의 인정 — 패배 시 조용히 카드 내려놓음 ("...허허, 시대가 바뀌었군")
  • 속내 감추기 — 표정·말투 절제 (포커페이스 한국식 = 무표정 + 침묵)

4-2. 도박 슬랭 (대사 작성 참고)

한국 슬랭 의미
"광 한 번" 인생의 한 번 / 큰 도전
"쩐주가 나오나" 큰 돈을 댈 사람 있나 / 베팅 압박
"다이" 폴드 (단호히)
"받겠소" 콜 (격조 있게)
"한 판 더" One more round
"오늘은 안 풀리네" 운이 없네
"이번 판은 내 거다" 이번엔 내가 이긴다
"장난해?" 패배 시 분노 / 의심
"허허, 잘 보셨네!" 블러프 들킨 광대 (탈춤 영감)
"...수고 많았소" 패배 후 격조 있는 인정 (남궁천)

→ 03-characters 대사 작성 시 이 표 참조.

4-3. 도박판의 어두운 면

  • 가산 탕진: 화투로 집 / 가족 / 인생 잃은 사람들
  • 중독: 도박 멈출 수 없음의 비극
  • 사기 (마패): 속임수 / 짜고 치기 / 카드 표시
  • 빚 / 폭력: 도박 빚으로 인한 사회 문제

→ K-Sutda 는 시각적 무게감 으로 이 정서 전달. 직접 묘사 ❌ (사행성 / 모방 회피).

5. 한국 미술 / 디자인 참고

5-1. 민화

  • 책가도: 책·꽃·과일 그림. 단정한 구도, 다색 사용
  • 화조도: 꽃·새 그림. 자연 묘사 + 상징
  • 민화 호랑이: 단순 라인 + 강한 표정
  • 십장생도: 장수 상징 10가지 동식물

→ 화투 카드 일러스트 (05-card-art-style) 참고.

5-2. 단청 (丹靑)

  • 한국 전통 건물 (사찰 / 궁궐) 채색 패턴
  • 짙은 청록 / 주홍 / 황금 조합
  • 기하학 + 자연 모티프

→ 카드 뒷면 / 배경 / UI 패턴 참고.

5-3. 글씨체 (서예)

서체 분위기
궁체 (宮體) 단정, 격조 (조선 왕실)
민체 자유, 개성
명조체 굵직, 무게감 (메인 헤더)
고딕체 모던, 가독성 (본문)

→ 로고 / 큰 텍스트에 명조 / 궁체 톤 권장.

5-4. 한지 텍스처

  • 거친 결 + 부드러운 톤 다양 + 약간의 얼룩
  • UI 패널 / 카드 뒷면 / 배경 매트 텍스처 활용

6-2. 음계

  • 한국 전통 음계 (정악): 평조 / 계면조 — 5음 (pentatonic)
  • 일본 5음과 다름 — 음정 간격 차이로 한국적 정서 구분
  • 단조 (Minor) 가 자연스러움

6-3. 참고 작품

  • 김덕수 사물놀이 — 4 타악기 합주
  • 이날치 + 앰비규어스 댄스 컴퍼니 — "범 내려온다" (한국 전통 + 모던 융합)
  • 이상철 (드라마 OST) — "각시탈" / "마패" 사운드트랙
  • 국립국악원 공식 음원 — 정악 / 민속악 다수 공개

7. 한국 사회의 한 단면

7-1. K-Sutda 의 메타 메시지 (옵션)

K-Sutda 는 단순 카드 게임이지만, 한국 도박 문화의 어두운 면도 게임 내 시각적으로 반영:

  • 명성치 한도 (-150) — 가산 탕진의 비유. 너무 깊이 가면 끝.
  • 블러프 시스템 — 한국 도박판의 "뻥" / 사기 / 속임수
  • 단계 7 탈춤 영감 — 가면 뒤 정체불명 (한국 가면 광대 = 사회 풍자)
  • 단계 8 남궁천 — 전설의 마패 = 한국 도박판의 신화

→ 단순 코믹 톤 ❌ → 진중함 + 한국적 정서 유지.

7-2. 윤리적 메시지

  • 게임 내 도박 미화 ❌
  • 명성치 = 가상 단위, 실제 환금성 없음 (모든 화면 disclaimer)
  • 게임오버 시 "...내 차례는 여기까지인가" 와 같은 체념의 격조 — 도박 패배 = 인생 끝이 아닌 한 판의 끝

9. 디자이너 ↔ 한국 문화 컨설팅

9-1. 외국 디자이너 협업 시

  • 한국 화투 ≠ 일본 하나후다 — 반복 강조
  • 한국 의상 (한복) 일본 기모노와 구분 — 색·소매·옷고름
  • 한국 식물 (소나무 / 매화 / 단풍) 일본 식물과 묘사 다름
  • 한국 도박 슬랭 직역 ❌ — 톤 살리기

9-2. 한국 디자이너 협업 시

  • 자연스럽게 한국 정서 이해 — 디테일만 합의
  • 한국 영화·드라마 레퍼런스 적극 공유
  • 어두운 정서 (도박판) 톤 유지 — 가벼운 카툰 톤 회피

9-3. 외부 자문 (옵션)

  • 한국 화투 / 도박 문화 학자 자문 (옵션, 한국학)
  • 한국 전통 음악가 자문 (작곡가 외주 시)
  • 한국 의상·소품 자문 (캐릭터 디자인)

10. 결론 — K-Sutda 의 한국적 정체성

K-Sutda 는 한국 화투의 무게감을 잃지 않으면서
발라트로의 빌딩 재미를 결합한 게임입니다.

디자이너의 모든 결정은 다음 질문으로 검증:
1. "이건 한국적인가, 일본적인가?"
2. "이건 진중한가, 가벼운가?"
3. "이건 도박판의 정서를 담는가?"
4. "이건 한국 사용자가 공감할 수 있는가?"

답이 "한국적, 진중, 도박판 정서, 한국 공감" 이면 진행.
하나라도 의심이면 재검토.

11. 라이선스 / 저작권 — 한국 문화 자료

  • 한국 화투 자체 = public domain (수백 년)
  • 한국 전통 음악 (악기 / 음계) = 공유 자원, 특정 작곡 / 편곡은 저작권 보호
  • 한국 영화 / 드라마 = 저작권 보호 — 레퍼런스만, 직접 사용 ❌
  • 한국 민화 / 단청 = 대부분 public domain (조선 시대 작품) — 박물관 디지털 자료 라이선스 확인
