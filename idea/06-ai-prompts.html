0-4. 비율 처리 (정사각 PNG → 세로 카드)

⚠ Nano Banana 는 프롬프트와 무관하게 1024×1024 정사각 PNG 만 출력합니다 (현재 모델 제약). 게임 카드 슬롯은 220×320 세로 비율 (2.2:3.2).

Card.gd 자동 처리: _draw_texture_fill() 헬퍼가 정사각 PNG 를 카드 비율에 맞춰 중앙 crop (cover 모드) 합니다. 좌우 12~13% 가 잘리니, 가장자리에는 핵심 요소 (한자, 도장) 배치 금지.

디자이너 권고 안전 영역:

  • 좌우 가장자리 13% 는 잘릴 수 있음 → 핵심 모티프는 가운데 74% 안에
  • 상하 가장자리는 그대로 보존 (잘리지 않음)
  • 한자 "光" / 도장 / 식물명 한글 등 텍스트 요소는 가운데 영역에 배치

후처리로 PNG 를 220×320 직접 crop 해도 OK (그래도 코드는 동일 동작).

8. 프롬프트 조정 팁

결과가 일본 톤일 때

→ "Korean Hanafuda, NOT Japanese Hanafuda. Korean Minhwa folk painting style. NOT Ukiyo-e." 강조.

결과가 너무 모던할 때

→ "Traditional Korean folk painting (Minhwa), bold ink outlines, hand-painted feel" 강조.

결과가 너무 사실적일 때

→ "Stylized illustration, NOT photorealistic, NOT 3D render, hand-drawn folk art" 강조.

광패 임팩트 부족할 때

→ "Strong gold border (4px), golden halo around moon/sun, this is the most powerful card 'Gwang'" 강조.

일관성 (시리즈 카드) 유지

→ Nano Banana 의 "consistent character / consistent style" 기능 활용. 1번 결과를 reference 로 첨부 → "Same illustrator, same style, same color palette as reference."
