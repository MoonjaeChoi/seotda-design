⚠ 미팅 시 결정 보조 — Do / Don't

컬러 (Palette)

  • ✅ 한국 도박판 톤 (먹·금·한지·주홍)
  • ❌ 카지노 적색 / 형광 색 / 일본 사쿠라 분홍

톤

  • ✅ 진중함 + 한국 정체성
  • ❌ 가벼움 / 발라트로 직접 모방 / 도박 미화

화투

  • ✅ 한국 화투 (송학 / 단풍 / 모란)
  • ❌ 일본 하나후다 / 사실주의 / 카툰
