▶ 옵션 C — 광패 일러스트 + 슬로건 (영화 포스터 톤) 채택

  • 구조: 상단 광패 2장 일러스트 → 큰 로고 → 슬로건 → CTA 버튼 (새 런 시작) → 보조 링크
  • 로고 사이즈: 72px (현재 mockup) 또는 96px (Palette.FONT_TITLE) — 디자이너 시안에서 결정
  • 슬로건: "광 한 번 보는 데 인생을 건다." — 항상 표시
  • 광패 일러스트: 3월 벚꽃광 + 8월 공산광 (광땡 조합) 권장. 디자이너 일러스트 작업 후 통합
  • 버튼 위계: "새 런 시작" 강조 (큰 CTA 버튼, 금색 채움). 그 외 (이어하기 / 리더보드 / 확률 / 규칙) 작은 텍스트 링크
  • 닉네임: CTA 버튼 위 (옵션 C 그대로)
  • 배경: 짙은 먹 + 가벼운 황금 글로우 / 한지 결 텍스처 (디자이너 결정)
  • 폰트: Title = Noto Serif KR Bold (✅ 결정), Body = Pretendard

📌 다음 작업: 디자이너 → MainMenu 영화 포스터 시안 (광패 일러스트 포함). 광패 일러스트는 화투 카드 작업 (Tier 1) 과 병행.
