첫 미팅 의제 5. K-Sutda 의 핵심 차별화 — 1차 (주홍, 작은 결정) vs 2차 (금색, 본격 베팅) 색 코드.

1. 1차 베팅 모드 (🃏 First Bet)

발동 시점: 첫 카드 1장 받은 직후. 작은 commit (5명성치) — 다이도 -5만 손실.
🃏 1차 베팅 (콜 5) 명성치: 245
색 의도: 주홍 보더 = 위험 / 결단 / 한국 도박판 톤. "다이 (-5)" 명확한 손실 표시.

발동 시점: 두 번째 카드 받은 후. 본격 베팅 (콜/레이즈/올인/폴드 모두 가능).

3. 명성치 위험 영역 표시

명성치 < 0 (위험) / < -100 (게임오버 임박). prestige 라벨 빨강 + 펄스 (모션 M-07).
🃏 1차 베팅 (콜 5) ⚠️ 명성치: -120 (게임오버 임박: -150)
잔액 부족 시 콜 / 레이즈 비활성 (alpha 0.4). 다이만 활성. 게임오버 위험 시각 강조.

✅ 결정 항목 — BettingPanel

  • 1차 vs 2차 색 구분: 주홍 (1차) ↔ 금색 (2차) — 충분히 구분되나? 더 강한 시각 차이 필요?
  • 이모지 사용: 🃏 (1차) / 🎴 (2차) — 정식 아이콘 (SVG) 으로 교체?
  • 다이 손실 표시: "다이 (-5)" — 손실 즉시 노출 (현재) vs 단순 "다이" (별도 hint)?
  • 1차 레이즈 fixed: 항상 +5 (현재) — 옵션 (사용자 선택) 도 가능?
  • 2차 슬라이더: 항상 표시 (현재) vs 사용자 "레이즈" 클릭 후 나타남?
  • 버튼 4 상태: default / hover / pressed / disabled 시각 차이 충분?
  • 모바일: 4 버튼 가로 배치 (현재) vs 2×2 격자? (12-mobile-ui 참조)
  • 위험 영역 강조: 명성치 라벨만 (현재) vs 전체 패널 빨강 글로우?
