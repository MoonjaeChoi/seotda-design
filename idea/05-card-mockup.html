✅ 결정 항목 — 화투 카드 톤

  • 변형 선택:
    • A (현재 코드) — 베타 직전 즉시 / 정식 출시 후 교체
    • B (한국 민화) — 정식 작업 권장, 시간 ↑ + 비용 ↑
    • C (모던) — 글로벌 친화, 한국 화투 친숙도 ↓
    • 혼합 — 광패만 B, 그 외 A 등
  • 광패 강조 강도: 보더 두께 (2 / 3 / 4px) + 글로우 강도 결정
  • type 라벨: 한글 "광 / 띠 / 열 / 피" (A/B) vs 한자 "光 / 帶 / 列 / 皮" (C)?
  • 월 표기: "1월" (A/B) vs "1月" (C)?
  • 식물명 표시: "송학" (A) vs 일러스트만 (B) vs "松鶴" 한자 (C)?
  • 샘플 작업 의뢰: 1월 송학광 + 3월 벚꽃광 2장 먼저 → 톤 합의

📌 추천: 베타엔 A 그대로 (즉시 사용 가능) → 베타 피드백 후 B 양산 의뢰 (정식 출시 직전). 디자이너 의견 듣고 결정.
