13b — psychological_pressure Toast VFX

단계 7 「각시탈의 미소」 심리 압박 토스트 시각 명세. docs/17-effect-dispatch-extensions §3-4 Q2 후속 deliverable.

0. 역할

psychological_pressure effect는 score/mult 영향 없음 — UI overlay 만. 플레이어가 가짜 베팅 위협에 심리적 압박을 느끼도록 설계.

핵심 원칙: 실제 점수 폭발 토스트(금색)와 시각적으로 명확히 구분.

1. 레이아웃 구조

┌──────────────────────────────────┐
│  🎭 각시탈의 미소                │  ← 마패 이름, font_size 13 뮤트 핑크
│         ⚠  +500                  │  ← 가짜 칩, font_size 34 선명한 빨강
└──────────────────────────────────┘
  • 배경 패널: 어두운 적색 반투명 Color(0.25, 0.02, 0.02, 0.88)
  • 테두리: 2px 적색 선 Color(0.85, 0.10, 0.10, 0.80)
  • 모서리: 4px radius

2. 색상 코드 (GDScript)

요소 Color 설명
배경 Color(0.25, 0.02, 0.02, 0.88) 어두운 적색 반투명
테두리 Color(0.85, 0.10, 0.10, 0.80) 밝은 적색 선
마패 이름 Color(0.75, 0.45, 0.45) 뮤트 핑크레드
칩 숫자 Color(1.00, 0.12, 0.12) 선명한 빨강

구분 근거: 실제 점수 토스트는 금색(#F3C72E) / 명성치 변화는 연녹색 / 적색은 「경보·가짜」 전용.

3. 폰트 크기

요소 font_size 비고
마패 이름 (🎭 각시탈의 미소) 13 부가 정보, 작게
칩 숫자 (⚠ +500) 34 위협적이되 실제 점수보다 작음

docs/18 점수 폭발: font_size 40 금색 → 이 토스트는 34 적색으로 차등.

4. 애니메이션 타임라인

구간 시간 내용
Phase 1 Pop-in 0 → 0.35s scale 0.4 → 1.08, opacity 0 → 1 (TRANS_BACK EASE_OUT)
Phase 2 Settle 0.35 → 0.50s scale 1.08 → 1.00
Phase 3 Drift 0.50 → 2.20s position.y −45px 상승 (TRANS_QUAD EASE_OUT)
Phase 4 Fade 2.00 → 2.50s opacity 1 → 0
Cleanup 2.60s queue_free()

총 지속: 2.6초 (기존 2.1초 → 0.5초 연장, 읽기 시간 확보).

5. 위치

  • 기준점: ai_card_a.global_position + Vector2(-20, -80)
  • Phase 3 동안 y 방향으로 −45px 추가 상승
  • 패널 pivot: panel.size / 2 (center — scale 기준점)

6. 텍스트 포맷 (_fmt_pressure_chips)

칩 숫자는 K/M suffix 약식:

원값 표시
500 +500
1200 +1.2K
5000 +5K
1500000 +1.5M

Board._fmt_pressure_chips(chips: int) -> String static 헬퍼 담당.

7. 학습 정책 (docs/17 §3-5)

  • 단계 7 첫 만남 → 표시 안 함 (SaveManager.was_boss_defeated(7) == false → early return)
  • 단계 7 패배 후 다음 런부터 정상 표시 (로그라이크 "a-ha 모먼트")

8. 구분 대조표

토스트 종류 색 font_size 목적
점수 폭발 (docs/18) 금색 #F3C72E 40 실제 점수
명성치 변화 연녹 / 적 24~28 명성치 갱신
심리 압박 (이 토스트) 적색 #FF1F1F 34 가짜 위협
올가미 raise 경보 주황 20 AI 베팅 신호
