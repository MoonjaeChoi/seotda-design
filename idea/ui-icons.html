K-Sutda — UI 아이콘 프롬프트

UI 아이콘 10종 + 앱 아이콘 생성 가이드. 아이콘은 단순 형태가 많아 Figma/Illustrator 직접 제작이 AI 생성보다 효율적인 경우가 많음.

0. 개요

0-1. UI 아이콘 사양

항목 값
마스터 해상도 64 × 64 px (→ 게임 내 32 × 32 표시)
형식 PNG (투명 배경) 또는 SVG
색 단색 흰색 #FFFFFF 또는 금색 #F3C72E — 배경 투명
저장 경로 assets/ui/{파일명}.png (또는 .svg)
선 두께 3 px 기준 (64px 기준), NO 그라데이션, NO 그림자

0-2. 스타일 원칙

  • 한국 전통 문양에서 모티프 차용 — 전통 + 현대 플랫 아이콘 혼합
  • Bold line icon — 라인 아트, 두꺼운 선
  • 단색 — 흰색 1 레이어 또는 금색 1 레이어 (배경 투명)
  • 32×32 px 에서도 식별 가능 — 지나친 디테일 금지
  • 상세 spec: design-handoff/02-asset-spec.md §3-3

1. 베이스 아이콘 스타일 가이드

1-1. Nano Banana 생성 시 공통 베이스 프롬프트

Single flat icon for a Korean card game UI.
64×64 px, white on transparent background.
Bold line icon style — thick outlines, minimal detail, NO fill gradients.

[Style]
- Korean traditional + modern flat icon hybrid
- Bold black outlines (3px weight at 64px)
- White icon on transparent background
- Think: app icon / game HUD button
- NOT decorative illustration — FUNCTIONAL icon (must read at 32×32 px)
- NOT 3D, NOT drop shadow, NOT photorealistic

[Motif]
- {MOTIF_DESCRIPTION}
- Inspired by Korean traditional folk symbol or design motif

[Specifications]
- 64×64 px square output
- Transparent background (PNG)
- Pure white #FFFFFF icon only
- Clean crisp edges — flat color, no anti-alias blur
- Safe zone: 6px from all edges (8px total inner safe area)

Negative: color fill (white outline only), photorealistic, 3D render,
gradient, shadow, Japanese aesthetic, text labels, multiple icons in one image,
complex decoration, thin fragile lines (must be bold)

1-2. Figma/Illustrator 직접 제작 권장

단순 기하 형태 아이콘은 AI 생성보다 직접 제작이 더 정밀하고 빠름:

권장 직접 제작 아이콘: icon_1st_bet, icon_2nd_bet, icon_danger,
                       icon_calendar, icon_leaderboard, icon_settings

Figma 제작 팁:
1. 64×64 px 아트보드
2. 전통 문양 레퍼런스 검색 (Google: "한국 전통 문양 벡터")
3. Auto Layout으로 정렬
4. Export as PNG @1x (64px) and @2x (128px)
5. iOS에서 1x = 32px, 2x = 64px 로 자동 처리

2. 아이콘 10종 개별 프롬프트

2-1. icon_bluff.png — 블러프 가면

용도: 블러프 action 버튼 (BettingPanel)

Korean traditional MASK (양반탈 / Yangban mask) face icon.
Flat white line icon, 64×64 px, transparent background.

[Motif]
- Simplified face of a Korean Yangban mask (양반탈)
- Round face outline, stylized eyes (curved slits), curved smile
- Abstract mask holes for eyes (NOT detailed — just the essential lines)
- Small mask string detail at sides (optional — only if readable at small size)

[Style]
- Bold line icon — mask face reduced to essential strokes
- The smile should read as "knowing" or "scheming" — curved upward
- 5-6 lines maximum — this is an icon, not an illustration

Negative: full theatrical illustration, Halloween horror, color fill,
Japanese Noh mask shape (Korean only), detailed shading

Figma 대안:

Shape: Circle (face) + 2 eye arcs (curved rectangles) + 1 mouth arc
Traditional reference: 양반탈 코드 이미지 (공공 도메인)
Line weight: 3px stroke

3. 앱 아이콘 (icon_1024.png)

📁 저장 경로: assets/marketing/icon_1024.png

3-1. iOS App Store + Android Google Play 공용

K-Sutda mobile app icon — 1024×1024 px, PNG.
Korean gambling roguelike card game icon.

[Motif]
- TWO hwatu gwang cards (3월 cherry blossom gwang + 8월 moon gwang)
  slightly overlapping, fanned upward from bottom-center
- The "光" character visible on both cards — golden
- Above the cards: subtle golden glow / radiance
- Below/behind cards: "K" initial or subtle "韓鬪" Korean logo mark
  (just an outline impression — text is added in code)

[Style]
- Rich, deep background #0D2619 (mandatory — no white/light background for app icons)
- Cards have gold #F3C72E gwang border and glow
- Bold, impactful — must be recognizable at 60×60 px (iPhone small icon)
- NOT anime, NOT casual mobile art — matches the game's cinematic tone

[Colors]
- Background: deep ink #0D2619
- Cards: hanji beige #F5EBD1 with gold "光" marks
- Card glow: radiant gold #F3C72E
- Optional accent: subtle vermillion #D93333 edge

[Specifications]
- 1024×1024 px square
- Composition centered
- Corner radius NOT needed (added by OS automatically)
- Must read clearly at: 1024px, 512px, 180px, 120px, 60px
- CRITICAL: at 60px, only the 2 cards + glow should be visible (no fine detail)

[Test sizes — check readability]
- 1024px: full detail visible
- 180px: cards + glow clearly visible
- 60px: golden glow + 2 card shapes = brand identity maintained

Negative: white background, anime art, K-pop idol, Western card ace,
casino chip, complex background (must be deep solid ink), multiple small elements

4. Figma/Illustrator 직접 제작 가이드 (권장)

4-1. 아이콘 세트 일관성 규칙

모든 아이콘 공통 설정 (Figma Style):
- Canvas: 64×64 px
- Safe zone: 6px margin (inner icon area: 52×52 px)
- Stroke weight: 3px (outer) / 2px (inner detail)
- Stroke cap: Round
- Color: #FFFFFF (단색 흰색), 배경 투명
- Corner radius: 2px on rectangular shapes
- Export: PNG @1x (64px), @2x (128px), @3x (192px)

4-2. 전통 문양 레퍼런스 소스

  • 국립중앙박물관 e뮤지엄 오픈 컬렉션 (공공 도메인)
  • 한국문화재청 문화유산 디지털 자료
  • 국가기록원 기록물 (공공 도메인)
  • 검색어: "한국 전통 문양", "단청 문양 벡터", "민화 문양"

5. 검증 체크리스트

UI 아이콘 (10종)

각 아이콘 검증:
[ ] 64×64 px에서 선 명확한가?
[ ] 32×32 px로 축소 시 여전히 읽히는가?
[ ] 배경 투명인가 (PNG)?
[ ] 색: 흰색 단색인가 (또는 금색 — 특별 용도)?
[ ] 선 두께 충분한가 (너무 가늘지 않은가)?
[ ] 한국 전통 모티프 요소가 포함되었는가?
[ ] 다른 아이콘과 스타일 일관성 있는가?

앱 아이콘 (icon_1024.png)

[ ] 1024×1024 px 저장?
[ ] 60px 축소 시 읽히는가?
[ ] 배경 완전히 불투명한가 (앱 아이콘은 투명 배경 불가)?
[ ] 어두운 배경 (#0D2619)?
[ ] 광패 2장 + 금색 글로우 가시적?
[ ] iOS 모서리 라운드 crop 시 핵심 요소 잘리지 않는가?
     (1024px 기준 모서리 약 160px 라운드 — 핵심 요소 중앙에)
