K-Sutda — 스토리텔링 / 백스토리

03-characters 캐릭터 시나리오의 확장판. 8 보스 깊은 백스토리, 메타 서사, 컷신 시나리오, 한국 도박판 세계관.

0. 스토리텔링 원칙

0-1. 톤

  • ✅ 무겁고 진중함 — 한국 도박판의 그림자
  • ✅ 함축적, 짧음 — 영화 "마패" 의 절제된 대사 톤
  • ✅ 인물 중심 — 게임 메커니즘이 아닌 사람의 사연
  • ❌ 코미디 / 가벼움
  • ❌ 장황한 설명 (보여주기 > 말하기)

0-2. 게임 안에서의 스토리

  • 컷신: 최소 (단계 인트로 / 엔딩 정도)
  • 대사: 짧게 (한 줄, 무게감)
  • 시각: 캐릭터 일러스트 + 배경 + 소품으로 전달
  • 음향: 한국 전통 악기 + 침묵
  • 선택: 사용자 직접 행동 (베팅 / 다이) 이 곧 스토리

0-3. 사용자 해석 여지

K-Sutda 의 모든 캐릭터 / 사연은 열린 결말 / 다중 해석 가능:

  • 무명 (플레이어) 가 왜 노름판에 왔는가? → 빚 / 복수 / 호기심 (사용자 마음대로)
  • 남궁천 (단계 8) 이 왜 무명을 불렀나? → 후계자 찾기 / 죽음 직전 마지막 한 판 / 시험

→ "정답" 없음. 사용자가 자기 이야기 만들기.

1. 세계관 (World Setting)

1-1. 시대 / 공간

  • 시대: 현대 한국 (2020년대) — 단, 한국 전통 도박판 분위기 유지
  • 공간: 한국 어느 도시 / 산속 / 다실 — 구체적 지명 없음 (보편적)
  • 사회: 한국 도박 문화의 어두운 면 (가산 탕진 / 중독) 직접 묘사 X, 인물 사연으로 암시

1-2. 노름판 (場)

"노름판" = K-Sutda 의 세계.

평범한 사회 밖, 명성치 (名聲) 라는 가상 통화로 인생을 거는 곳.
8 단계의 노름판 — 각 단계마다 다른 인물 / 다른 분위기.

마지막 단계 = 전설의 마패 남궁천 의 다실.
이긴 자만이 "전설" 칭호를 받는다.

1-3. 명성치 (名聲) — 가상 통화의 의미

  • 실제 화폐 X → 명성치
  • "이긴다 = 명성을 쌓는다"
  • "잃는다 = 평판을 잃는다"
  • 게임오버 (-150 도달) = 사회적 명예 추락
  • → 도박 미화 회피 + 진중한 게임 톤

1-4. 마패 (打子) — 도박의 전설

  • "마패" = 한국에서 도박 고수를 부르는 단어
  • 영화 "마패" (2006) 의 무대
  • K-Sutda 의 단계 8 보스 "남궁천" = 살아있는 전설
  • 무명 (플레이어) 은 무명 → 전설로 성장

2. 메인 캐릭터 — 무명 (無名)

2-1. 백스토리

이름을 잃어버린 자.

3년 전, 평범한 회사원이었다.
어느 날 가족이 도박 빚으로 무너졌고,
무명은 노름판에 발을 들였다.

처음엔 빚을 갚기 위해.
그 다음엔 복수를 위해.
지금은 — 모른다.

"광 한 번 보기 위해."
무명이 말할 수 있는 유일한 이유.

얼굴을 가리는 갓 아래, 무명은 노름판을 떠돈다.
오늘은 8 단계의 마지막 노름판.
전설의 마패 남궁천이 그를 부르고 있다.

2-2. 비주얼

  • 얼굴 일부 가림 (갓 그림자)
  • 흑색 도포 (단계 진행에 따라 살짝 변화 가능)
  • 손에 화투 2장
  • 표정 없음 (포커페이스)

2-3. 대사 / 보이스

상황 대사
런 시작 (MainMenu 진입) "...한 판 더."
단계 1 클리어 (말 없음. 카드 정리)
단계 5 클리어 "...아직이군."
단계 7 클리어 "...마지막 판이 보인다."
단계 8 진입 (남궁천) "...오늘, 끝내겠소."
단계 8 클리어 (런 완료) "...광이 보였다." (작게 미소)
게임오버 "...내 차례는 여기까지인가."

2-4. 사용자 ↔ 무명

  • 사용자 닉네임 = 무명의 이름 (입력 시 대체)
  • 사용자의 베팅 = 무명의 결정
  • 사용자 게임오버 = 무명의 패배 (다음 런에서 다시 시작)

3. 8 보스 — 깊은 백스토리

3-1. 단계 1 — 박달삼 (40대 남, BASIC)

백스토리

동네 어귀 평상에서 매일 막걸리 한 잔.
20년 전 직장 잃은 후, 도박이 유일한 낙.
가족과는 연락 끊긴 지 오래.
큰 돈 안 거는 대신, 매일 진다.
"한 판 더!" 외치며 동전을 평상에 던진다.

무명은 박달삼을 본다 — 자기 미래의 한 모습.

손가락에는 금반지 — 남편 사별 후 유일한 추억.
"이건 봐 줄게, 한 번만이야."

백스토리

백스토리

산속 절에서 30년 수행.

한 때 그는 — 한국 도박판의 전설이었다.
"독사" 라 불렸다. 1월 + 4월 패의 명수.

30년의 침묵.

그런데 — 절을 짓기 위해 시주가 필요했다.
한 번만 더. 마지막 한 번만.
회색 승복 입은 채, 그는 다시 노름판에 나타났다.

"...인연이라 생각하시오."

3-7. 단계 7 — 탈춤 영감 (50~60대 남, PSYCHOLOGICAL)

백스토리

정체불명. 가명 다수.

탈 뒤의 얼굴을 본 사람이 없다.

20년 전 어느 도박판에서 처음 등장.

"허허, 한 판 받아 보시게."

어떤 자는 그가 전설의 마패 남궁천의 옛 친구라 한다.
어떤 자는 그가 죽은 자의 영혼이라 한다.

확실한 것 하나 — 그는 늘 웃고 있다.
탈 뒤에서.

3-8. 단계 8 — 남궁천 (70대 남, COMPLEX)

백스토리

한국 도박계의 전설.

50년 전부터 그는 모든 마패의 스승.
"남궁천 앞에서 패를 본 자 = 자기 패를 본 적이 없는 자."

20년 전, 그는 갑자기 사라졌다.
산속 다실에 자리를 잡고 — 침묵.
누구의 도전도 받지 않았다.

3개월 전, 그는 한 통의 편지를 보냈다.
"무명이라는 자를 만나 보고 싶다."
무명은 그가 누군지 알지 못한다.
그런데 왜 자기를?

다실의 문을 열면 — 남궁천이 앉아 있다.
백발, 검은 한복, 정갈한 손가락.
다구 옆에 화투 2장이 놓여 있다.

"오랜만에... 진짜 한 판이군."

4. 메타 서사 (Meta Narrative)

4-1. 한 런의 서사 흐름

[프롤로그]
무명이 노름판에 발을 들이다.
"...한 판 더."

[단계 1-2: 가벼운 시작]
박달삼, 오영자 — 동네 / 시장의 평범한 사람들.
무명은 자기 미래의 한 모습을 본다.

[단계 3-4: 본격 도전]
강도진의 자만, 최강수의 압박.
무명은 진짜 도박판에 들어선다.

[단계 5-6: 정신적 단계]
현묵 스님의 절제, 한채린의 계산.
무명은 깨닫는다 — 도박은 카드가 아닌 마음.

[단계 7: 가면 뒤의 신화]
탈춤 영감의 블러프.
무명은 진실과 거짓의 경계에 선다.

[단계 8: 전설과의 마지막 한 판]
남궁천의 다실.
무명이 이긴다면 → 새 전설.
무명이 진다면 → 또 하나의 잊혀진 노름꾼.

[엔딩]
승리: "...광이 보였다."
패배: "...내 차례는 여기까지인가."

4-2. 단계 간 연결 (옵션)

각 단계 클리어 후 짧은 컷신 (5~10초):

[단계 3 클리어 후]
[강도진이 자기 카드를 던지며 일어선다.]
[무명은 카드를 정리한다.]
[단계 4 진입 — 최강수의 어두운 도박장]

→ 베타 후 콘텐츠. 정식 출시에 들어가면 좋음.

4-3. 단계 8 후 — 새 런

승리 시 메인 메뉴로 돌아가면:

  • 닉네임 옆에 "전설" 칭호
  • 다음 런 시작 시 — 무명이 다른 형태로 보일 수 있음 (옵션, 변형)
  • 메인 메뉴 배경 — 살짝 다름 (다실 느낌)

→ 메타 진행 (Persistent meta) 디자인.

5. 컷신 / 인트로 시나리오

5-1. 게임 첫 실행 — 프롤로그 컷신 (15~20초)

[화면: 검은 화면]

[음악: 가야금 단조 잔잔]

[자막 — 한 줄씩 등장, 각 3초 유지]

"한국 어느 도시의 밤."
↓
"노름판의 문이 열린다."
↓
"무명이라는 자가 들어온다."
↓
"...광 한 번 보기 위해."

[화면 페이드 → MainMenu]

→ 첫 실행 시 1회만. 스킵 가능.

5-2. 단계 8 진입 — 남궁천의 다실 (10초)

[Board 화면 진입 전]

[배경: 다실 문이 천천히 열린다.]
[음악: bgm_boss 의 가야금 솔로]

[남궁천의 모습 — 검은 한복, 백발, 화투 2장]

[자막]
"전설의 마패 남궁천."

[화면 전환 → Board (남궁천 일러스트)]

5-3. 런 클리어 — 새 전설 (15~20초)

[남궁천이 일어선다.]

[자막]
남궁천: "...자네가 새 전설이군."

[화면 페이드 → 한지 베이지]

[무명이 갓을 벗는다.]
[얼굴 = 사용자 닉네임 텍스트]

[자막]
"...광이 보였다."

[황금 비 + "전설" 칭호 알림]

[화면 → MainMenu (전설 칭호 표시)]

5-4. 게임오버 (-150 도달) — 침몰 (5~10초)

[Board 화면 어두워짐]

[자막]
무명: "...내 차례는 여기까지인가."

[화면 페이드 → 검은]

[MainMenu]

6. 캐릭터 간 관계 / 백스토리 연결

6-1. 가능한 연결

- 현묵 스님 (단계 5)
  ↔ 옛 이름 "독사" — 20년 전 도박판의 전설

- 탈춤 영감 (단계 7)
  ↔ 남궁천의 옛 친구? (탈 뒤의 정체)

- 남궁천 (단계 8)
  ↔ 무명을 부른 이유? (편지의 진의)

- 무명 (플레이어)
  ↔ 가족이 도박 빚으로 무너진 사연

6-2. 사용자 호기심 유도

직접 설명 X. 사용자가 게임 진행하며 발견:

  • 단계 5 현묵의 대사 중 "...독사 시절에..."
  • 단계 7 탈춤의 대사 중 "...옛 친구가 나를 기다린다오..."
  • 단계 8 남궁의 편지 (단계 7 클리어 후 옵션 컷신)

→ 베타 후 콘텐츠. 사용자 토론 / 추측 활성화.

7. 짧은 컷신 — 일러스트 + 자막

7-1. 일러스트 사양

  • 600 × 900 px (캐릭터 자체) 또는
  • 1920 × 1080 (배경 + 캐릭터 — 컷신용)
  • PNG 또는 .mp4 (애니메이션 짧은 모션)

7-2. 자막 사양

  • 폰트: KoPub바탕체 Bold 24~32px
  • 색: TEXT_PRIMARY (흰색)
  • 배경: 짙은 먹 + 알파 0.7 또는 그라데이션
  • 모션: fade-in 0.5s + 유지 3s + fade-out 0.5s

7-3. 음향

  • 가야금 / 대금 단음 (3~5초)
  • 침묵 권장 — 한국 도박판 톤

8. 엔딩 분기 (옵션)

8-1. 결말 1 — 새 전설 (메인 엔딩, 승리)

"...자네가 새 전설이군."
[전설 칭호]

8-2. 결말 2 — 게임오버 (-150)

"...내 차례는 여기까지인가."
[메인 메뉴로]

8-3. 결말 3 — 단계 N 실패 (단계 클리어 못함)

"...오늘은 안 풀리네."
[메인 메뉴로]

8-4. (옵션, 미래) 결말 4 — 평생의 노름꾼

사용자가 100+ 런 완료 시:
"...언제까지 이 판에 있을 것인가."
[숨겨진 엔딩 — 무명이 노름판을 떠나는 시나리오]

9. 사용자 ↔ 스토리 연결

9-1. 닉네임

  • 사용자 닉네임 = 무명의 정체성
  • 단계 8 엔딩에서 닉네임 노출

9-2. 시드 / 결정성

  • "같은 시드 = 같은 운명" — 사용자 끼리 도전 가능
  • 커뮤니티 시드 챌린지 (22-community)

9-3. 일별 활동

  • "오늘의 명성치" — 사용자의 일상
  • 일별 리포트 = 자기 인생의 한 단면 (DailyActivity)

10. 작업 우선순위

Tier 1 — 베타 직전 (필수)

  • 캐릭터 8명 기본 백스토리 (03-characters)
  • 메인 캐릭터 무명 — 짧은 백스토리 (HelpInfo 추가)

Tier 2 — 베타 운영 중

  • 단계 8 남궁천 등장 컷신 (10초)
  • 런 클리어 엔딩 컷신 (15초)
  • 단계 보스 8명 깊은 대사 (위 §3)

Tier 3 — 정식 출시

  • 첫 실행 프롤로그 컷신 (15초)
  • 게임오버 컷신 (5초)
  • 단계 간 짧은 전환 컷신 (선택)

Tier 4 — 후속 콘텐츠

  • 캐릭터 백스토리 연결 (현묵 = 독사, 탈춤 영감 = ?)
  • 숨겨진 엔딩 (100+ 런)
  • 시드별 특별 시나리오 (옵션)

11. 디자이너 / 작가 협업

11-1. 시나리오 작가

  • 8 보스 대사 (위 §3) 다듬기 + 추가
  • 컷신 자막 (한↔영 양쪽)
  • 단계 간 전환 (선택)

11-2. 일러스트레이터

  • 캐릭터 8명 + 무명 (03 + 14-cultural-guide)
  • 컷신 배경 (다실 / 노을 / 도박장 등)
  • 엔딩 / 게임오버 시각

11-3. 모션 디자이너

  • 컷신 짧은 모션 (자막 fade / 캐릭터 등장)
  • 황금 비 (런 클리어)

11-4. 영문 번역

  • 모든 대사 / 자막 영문 (10-localization)
  • 한국 도박 슬랭 톤 보존
  • 캐릭터 이름 → 영문 표기 (Dalsamee, Yeongja, Dojin 등)

12. 라이선스 / 윤리

  • 모든 캐릭터 / 스토리 = 본인 창작
  • 실존 한국 인물 / 영화 캐릭터 모방 ❌
  • 도박 미화 / 실제 도박 권유 ❌
  • 사회 약자 (도박 중독자 / 가족 피해자) 비하 / 풍자 ❌

13. 결론

K-Sutda 의 스토리는 게임 메커니즘 위에 얹는 장식이 아닌,
한국 도박판의 한 단면을 담는 그릇.

무명은 사용자 자신.
8 보스는 한국 사회의 다양한 도박꾼들.
남궁천은 전설 — 또는 사용자의 그림자.

매 런 = 한 인생의 압축.
승리 = 새 전설.
패배 = 또 한 명의 잊혀진 노름꾼.

스토리는 짧고 함축적.
사용자가 자기 이야기를 만들도록.
