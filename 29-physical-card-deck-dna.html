현황: 디지털 DNA 요약

현재 마패 카드 시스템은 완전히 디지털 전용으로 설계되어 있다.

구성 요소 수량 비고
화투 카드 40장 10월 × 4종(광·단·열·피)
마패 카드 120장 5등급, 8가지 효과 타입
합계 160장

현재 디지털 DNA의 핵심 가정들:

DNA 요소 현재 값 물리 제작 시 충돌 여부
배경색 #0D2619 진한 잉크 블랙 ⚠️ 잉크 도포율 ~80%, 고비용
텍스트·효과 런타임 코드 오버레이 ❌ 물리 인쇄에 존재 불가

Part 1. 반드시 해결해야 할 구조적 충돌 6가지

어느 방향을 택하든 이 여섯 가지는 결정이 필요하다.

충돌 1. 텍스트 처리

현재: 카드명·효과 텍스트·타이밍 정보가 모두 게임 코드가 런타임에 화면 위에 그린다.
물리: 모든 정보가 인쇄물에 직접 포함되어야 한다.

선택지:

  • A. 카드 하단 정보 밴드 추가 (CCG 구조 — MTG, 하스스톤 방식)
  • B. 텍스트 최소화 — 카드명만, 효과는 별도 룰카드 참조
  • C. 텍스트 없음 — 아이콘 전용 + 두꺼운 룰북

충돌 2. 보더 & 희귀도 표시

현재: 코드가 희귀도에 따라 보더 색을 자동 적용한다 (회색/청/자주/주황/주홍).
물리: 인쇄물에는 그 구분이 없다.

선택지:

  • A. 인쇄 후가공 — Common 없음 / Uncommon 은박 / Rare 금박 / Legendary 홀로그램 / Boss 빨간 포일
  • B. 카드 디자인에 희귀도 심볼 삽입 (단마패·삼마패·오마패·무쌍 아이콘)
  • C. 희귀도별 다른 카드 재질·코팅 (Common 무광 / Rare 유광 / Legendary 엠보싱)
  • D. 뒷면 색상으로 등급 구분 (단, 게임 중 정보 유출 문제)

충돌 3. 배경색

현재: #0D2619 — 잉크 도포율 약 80%.
물리: 고잉크 배경은 카드 뒤틀림, 인쇄비 증가, 잉크 번짐 위험.

선택지:

  • A. 유지 — 프리미엄 인쇄소, 고비용 감수
  • B. 차콜 그레이로 완화 — 도포율 ~50%로 낮춤
  • C. 밝은 한지 베이지로 반전 — 전통 인쇄 미학, 완전한 재설계
  • D. 카드 타입별 분리 — 화투는 밝음, 마패는 어두움

충돌 4. 색상 공간 (RGB → CMYK)

현재: 모든 색이 RGB 기준이다.
물리: 인쇄는 CMYK. 변환 시 주요 색상이 크게 달라진다.

문제가 되는 색들:

색 이름 RGB CMYK 근사 변질 수준
배경 #0D2619 진한 녹-흑 C60 M30 Y30 K90 중간 (재현 가능)
자주 #9B59B6 선명한 보라 C40 M65 Y0 K0 높음 (CMYK 보라는 탁함)
청명 #5BA4D9 선명한 하늘 C60 M20 Y0 K0 낮음
황금 #F3C72E 선명한 금 C0 M20 Y85 K0 낮음
연금 #9DB85A 연두빛 금 C40 M0 Y65 K15 낮음

→ 특히 bluff_synergy 강조색(자주 계열)이 CMYK에서 크게 탁해진다.

충돌 5. 카드 뒷면

현재: 존재하지 않는다.
물리: 게임에서 사용한다면 패 숨기기 기능 필수. 브랜드 정체성의 핵심 요소.

선택지:

  • A. 화투 전통 문양 재해석 — K-Sutda 세계관 요소 삽입
  • B. 단청 운문 패턴 다크 버전
  • C. 마패(馬牌) 문양 — 말 모티프 중심
  • D. 미니멀 — 로고 + 단색 배경
  • E. 화투용·마패용 뒷면 각각 다르게

충돌 6. 제품 범위

현재: 화투 40장 + 마패 120장 = 160장.
물리: 160장은 일반 보드게임 덱(52~110장)보다 많다.

선택지:

  • A. 전량 — 160장 완전판
  • B. 화투 40장만 — 아트 덱 재해석
  • C. 마패 큐레이션 — 희귀도별 대표 카드 (예: 각 등급 10장씩 = 50장)
  • D. 스타터 세트 — 40+30장 구성으로 게임 가능한 최소 세트
  • E. 모듈형 — 기본판(화투 40장) + 마패 확장팩 별도 판매

Part 2. 디자인 방향 — 6가지 관점

아래 6가지 의견은 방향의 다양성을 탐색하기 위한 것으로, 어느 것도 최종 결정이 아니다.

관점 A. 디지털 유산 보존

"다크 배경을 포기하지 말 것. 비용을 감수하고 프리미엄으로 가라."

K-Sutda의 시각 정체성은 #0D2619 위에 세워져 있다. 이것을 바꾸면 "다른 게임의 카드"가 된다. 고급 인쇄물 시장(Kickstarter, 보드게임 긱)에서는 오히려 다크 배경이 프리미엄의 기준이다.

제안하는 DNA 변경:

항목 현재 변경안
배경색 #0D2619 RGB CMYK 재정의: C60 M30 Y30 K90
텍스트 코드 오버레이 카드 하단 정보 밴드 (배경 #1A3020)
보더·희귀도 코드 자동 색 후가공: Common 없음 → Legendary 홀로그램
카드 뒷면 없음 단청 운문 패턴 다크 버전
색상 공간 RGB 처음부터 CMYK로 재지정
카드 크기 디지털 비율 타로 사이즈 70×120mm 권장

장점: 브랜드 100% 일관성, 강력한 비주얼 차별화, 프리미엄 포지셔닝
단점: 인쇄 단가 2~3배, 최소 발주 수량(MOQ) 높음, POD 불가
적합 제품: Kickstarter 한정판 콜렉터 덱

관점 B. 한국 전통 인쇄 미학

"물리 카드는 다른 매체다. 한지·목판화로 완전히 재해석하라."

화투 자체가 원래 밝은 배경의 물리 카드다. 디지털에서 흉내 냈던 "먹과 한지 느낌"을 물리에서 진짜로 구현하는 역전이 가능하다.

제안하는 DNA 변경:

항목 현재 변경안
배경색 #0D2619 잉크 블랙 한지 베이지 #F5EBD1
모티프 톤 밝은 모티프 / 어두운 배경 짙은 먹 모티프 / 밝은 배경 (양음 전환)

관점 C. 보드게임 산업 표준

"제조·유통의 표준을 따르면 모든 것이 쉬워진다."

Stonemaier, CMON, BGG Store 등 메이저 보드게임 유통 채널에는 컴포넌트 표준이 있다. 포커 사이즈(63×88mm), 슬리브 호환성, CMYK 가이드라인, 최소 텍스트 크기 등. 이 표준을 맞추면 제조 견적, 글로벌 유통, Kickstarter 풀필먼트 업체 선택이 모두 쉬워진다.

제안하는 DNA 변경:

카드 레이아웃 구조 (CCG 방식):

관점 D. 이중 DNA 공존

"디지털과 물리는 같은 세계관을 공유하되, 각자의 매체 언어를 쓴다."

발라트로의 디지털 UI와 굿즈·카드 상품은 시각적으로 다르다. 이것이 정상이다. "K-Sutda 물리 덱"은 디지털 게임의 복제품이 아닌 같은 세계관의 다른 표현으로 포지셔닝한다.

두 DNA의 분기 지점:

요소 디지털 DNA 물리 DNA
배경 #0D2619 잉크 블랙 낡은 한지 #D4C4A0
선 스타일 균일한 굵은 윤곽선 불균일한 목판화 느낌 선
색 팔레트 8가지 효과 타입 강조색 조선 오방색 계열로 재매핑
텍스트 코드 오버레이 목판 인쇄체 느낌 한글 활자
보더 코드 자동 희귀도 색 매듭(馬牌) 문양 인쇄 보더
아이덴티티 "K-Sutda 게임 UI" "조선 암행어사의 마패 세트"

절대 변하지 않는 공유 요소:

  • 카드명 (한국어 표기)
  • 모티프 주제 (같은 모티프, 다른 배경 위에)
  • 10개 모티프 군 체계 (G1~G10)
  • 마패 등급 세계관 (단마패·삼마패·오마패·무쌍)
  • 세트 보너스 마이크로 모티프

장점: 두 제품이 서로 강화, 물리 덱이 디지털 게임의 문화 상품이 됨
단점: 아트워크 두 벌 제작 필요, 일관성 관리 복잡도 증가
적합 제품: 디지털 게임 출시 기념 굿즈 + 독립 보드게임 동시 운영

관점 E. 화투 재창조

"K-Sutda의 핵심은 화투다. 물리 덱 = 재해석된 화투 덱 그 자체."

세상에는 화투를 재해석한 수십 종의 아트 덱이 있다. 하지만 K-Sutda 세계관의 스토리텔링이 담긴 화투는 없다. 화투 40장을 K-Sutda 로그라이크 세계관으로 재창조하면 게임 컴포넌트이자 독립 아트 상품이 된다.

각 화투 카드 타입별 세계관 레이어:

화투 타입 K-Sutda 레이어
광(光) 카드 (12장) 해당 월 보스 캐릭터 실루엣 삽입
단(段) 카드 (10장) 해당 월 연결 마패 카드 모티프 힌트
열(列) 카드 (10장) 족보 힌트 삽화 (끗 숫자 암시)
피(皮) 카드 (20장) 배경 세계관 요소 미니멀 표현

DNA 변경:

항목 변경안
배경 화투 전통 = 밝은 배경 유지 (월별 고유 색)
카드 크기 전통 화투 2배 확대 — 108×68mm (실용적)
보더 월별 고유 컬러 보더 (1월=청, 3월=분홍, 8월=은 등)

제품 범위 화투 40장 중심, 마패는 별도 확장

장점: 진정성 있는 한국 문화 상품, 화투를 아는 사람에게 강한 감동, 컴팩트
단점: 마패 시스템 120장을 물리로 표현 불가, 별도 룰북 없이 게임 불가
적합 제품: 문화 상품, 해외 K-culture 굿즈, 미술관·문구점 유통

관점 F. 프리미엄 오브제

"카드 게임이 아니라 수집품(Object)으로 포지셔닝하면 모든 제약이 기회가 된다."

160장은 보드게임 컴포넌트로는 너무 많지만, 럭셔리 아트 덱(타로 덱, 아티스트 플레잉 카드)으로는 적당한 수량이다.

DNA 변경:

항목 변경안 이유
배경 #0D2619 유지 이 포지셔닝에서는 오히려 고급감의 근거
텍스트 카드명만, 효과 없음 별도 룰카드 참조
보더 없음 (풀 블리드) 단, Legendary/Boss는 금박 테두리
카드 크기 타로 사이즈 70×120mm 일러스트 감상 최적
카드 뒷면 전체 동일 통합 디자인 게임용 아님 — 패 숨기기 불필요

후가공 등급 체계:

희귀도 후가공 단가 영향
Common 49장 무광 코팅 기준
Uncommon 36장 부분 유광 스팟 UV +15%
Rare 22장 엠보싱 +30%
Legendary 7장 홀로그램 포일 +80%
Boss 6장 레드 포일 + 엠보싱 +80%

패키징: 에나멜 슬리브 케이스 또는 마패 형태 케이스

장점: 고마진, 소량 생산 가능, 아트 커뮤니티 어필, 제2의 수익원
단점: 보드게임으로 사용 불가, 핵심 타겟과 어긋날 수 있음
적합 제품: Kickstarter 리워드, 게임 출시 기념 한정판, 굿즈샵

Part 3. 결정 전 공통 질문 5가지

어느 방향을 택하든 이 다섯 가지 질문에 먼저 답해야 한다.

Q1. 이 제품의 목적은 무엇인가?

선택지 설명
(A) 디지털 게임 홍보·굿즈 디지털 게임 구매자에게 물리적 보조 소품 제공
(B) 독립 물리 보드게임 디지털 게임 없이도 완전히 플레이 가능한 독립 게임
(C) 수집용 아트 오브제 게임 가능 여부 부차적, 소장 가치가 핵심

→ 결정: B — 독립 물리 보드게임. 룰북 동봉, 디지털 게임 불필요.

Q2. 어떤 카드를 포함하는가?

선택지 구성 장수
(A) 화투 40장만 아트 덱 재해석 40장
(B) 마패 120장만 보조 덱 120장
(C) 화투 40 + 마패 큐레이션 각 등급 대표작 ~85장
(D) 스타터 세트 게임 가능 최소 구성 ~70장
(E) 완전판 화투 + 마패 전량 160장
(F) 모듈형 기본판 + 확장팩 분리 유동적

→ 결정: C — 화투 40장 + 마패 큐레이션 ~45장 = 약 85장 (세부 선정은 Part 4 참조)

Q3. 게임 가능 여부?

수준 설명
Level 0 감상만 — 게임 룰 없음
Level 1 기존 섯다 룰로 플레이 가능 (화투만)
Level 2 K-Sutda 보드게임 버전으로 완전 플레이 가능
Level 3 디지털 게임과 연동 (QR 코드 등)

→ 결정: Level 2 — K-Sutda 보드게임 룰북 동봉. 물리 카드만으로 완전한 게임 성립.

Q4. 언어 정책?

선택지 적합 시장
한국어 전용 국내 시장
한국어 + 영어 병기 아시아 주요 시장
영어 전용 글로벌 보드게임 시장
3개 언어+ 글로벌 K-culture 팬

→ 결정: 국내전용 — 한국어 전용. 카드·룰북 모두 한국어.

Q5. 생산 방식?

방식 최소 수량 단가 리드타임 리스크
POD (MakePlayingCards 등) 1장~ 높음 2~3주 없음
디지털 단기 인쇄 50~500장 중간 3~4주 낮음
오프셋 단기 500~2000장 낮음 6~8주 중간
Kickstarter 오프셋 1000장+ 최저 3~6개월 높음

Part 4. 결정 확정 — 물리 보드게임 DNA 명세

Q1=B · Q2=C · Q3=Level2 · Q4=국내전용 · 관점 C (보드게임 표준) 기준으로 도출

4-1. 제품 정의

항목 확정

4-2. 카드 구성 (~85장)

종류 수량 선정 기준
